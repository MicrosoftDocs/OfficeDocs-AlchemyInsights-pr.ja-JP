# <a name="cant-create-a-team-in-teams"></a>Teams 内にチームを作成できない

Teams 管理者がチームの作成に必要な権限を付与していない可能性があります。  

Microsoft 365 で組織のセキュリティ グループの設定を作成または変更する方法については、管理者にお問い合わせください。 セキュリティ グループに追加されると、チームを作成することができます。
