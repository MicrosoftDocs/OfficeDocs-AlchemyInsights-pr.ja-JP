# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: '変更の処理中' のトラブルシューティング

長時間にわたり "変更の処理中" と表示される場合は、オンラインのファイルが開かれていたり、キュー内に多くのファイルが滞っていたり、非常に大きなファイルの同期が行われていたりする可能性があります。[OneDrive が処理中に停止する](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd)を参照してください。

"変更の処理中" のトラブルシューティングについては、この記事をお読みください。Outlook と同様に、PST ファイルまたはデータベースのような常に変更されているファイルがある場合にも、このメッセージが表示されます。 OneDrive で定期的に使用している[Outlook データ ファイルの移動](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e)方法を確認してください。
