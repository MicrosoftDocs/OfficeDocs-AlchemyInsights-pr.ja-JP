# <a name="sending-email-as-the-shared-mailbox"></a>共有メールボックスとしてメールを送信します。

共有されているメールボックスが作成されると、メールボックスのメンバーであるユーザーは共有メールボックス id (たとえば、 *info@contoso.com* ) として電子メールを送信することにします。手順については、次のように表示されます。 
  
[Outlook 2016 および Outlook 2013 で共有メールボックスを開いて使用する](https://support.office.com/en-us/article/open-and-use-a-shared-mailbox-in-outlook-2016-and-outlook-2013-d94a8e9e-21f1-4240-808b-de9c9c088afd)
  
[共有されているメールボックスを web 上で Outlook に追加します。](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)
