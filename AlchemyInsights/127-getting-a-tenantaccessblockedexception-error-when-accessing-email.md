# <a name="getting-a-tenantaccessblockedexception-error-when-accessing-email"></a>メールにアクセスしようとしたときに TenantAccessBlockedException エラーが表示される場合

このエラーが表示される理由はいくつかあります。次に例を示します。
  
- Office 365 管理センター ([https://portal.office.com/adminportal/home#/subscriptions](https://portal.office.com/adminportal/home#/subscriptions)) の [**課金**] の [**サブスクリプション**] ページで、サブスクリプションが有効であることを確認します。
    
- Office 365 管理センター ([https://portal.office.com/adminportal/home#/billoverview](https://portal.office.com/adminportal/home#/billoverview)) の [**課金**] の [**課金内容**] ページで、期限が過ぎた通知があるかどうかを確認します。
    
- アカウントの課金内容とサブスクリプションが最近更新された場合、その変更が反映されるまで時間がかかる可能性があります。数分後にもう一度試してみてください。それでも引き続き問題が発生する場合は、お問い合わせください。
