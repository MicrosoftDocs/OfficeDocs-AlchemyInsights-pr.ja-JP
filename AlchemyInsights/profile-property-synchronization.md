# <a name="profile-property-synchronization"></a>プロファイル プロパティの同期

Azure Active Directory からユーザー プロファイル アプリケーションを同期するのには AD のインポートの追加のプロパティをマップすることはできません。AD のインポートでは、あらかじめ構成された一連のタイマー ジョブの一貫したパフォーマンスを保証するためにプロパティに限定されます。
  
[同期プロファイルのプロパティの詳細については](https://go.microsoft.com/fwlink/?linkid=875671)
