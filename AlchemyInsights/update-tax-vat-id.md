# <a name="update-taxvat-id"></a>税 ID または VAT ID を更新する

税 ID または VAT ID に関連する問題がある場合は、Microsoft のライセンスパートナー (パートナーまたはソフトウェア アドバイザー) にお問い合わせください。 お客様の税 ID または VAT ID に関連する問題は、地域オペレーション センター (ROC) に連絡されます。
