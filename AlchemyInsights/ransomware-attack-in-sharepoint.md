# <a name="ransomware-attack-in-sharepoint-or-onedrive"></a>SharePoint または OneDrive でのランサムウェア攻撃

1.  OneDrive または SharePoint または Teams で悪意のあるファイルが検出された場合の対処方法については、「悪意のあるファイルが検出され[た場合](https://support.office.com/en-ie/article/what-to-do-when-a-malicious-file-is-found-in-sharepoint-online-onedrive-or-microsoft-teams-01e902ad-a903-4e0f-b093-1e1ac0c37ad2)の対処方法」を参照してください。
2. ランサムウェアからデータを保護する方法については、以下を参照してください。
    - [SharePoint と OneDrive でデータを保護する](https://docs.microsoft.com/sharepoint/safeguarding-your-data) 
    - [ランサムウェアの脅威に対する保護](https://docs.microsoft.com/windows/security/threat-protection/intelligence/ransomware-malware)    

3.  OneDrive または SharePoint 上のすべてのファイルは、ランサムウェアまたはマルウェアまたは暗号化ウイルスによって影響を受けますか? 

[OneDrive 同期] (存在する場合) または [エクスプローラービューを切断する] を、影響を受けるコンピューターからライブラリにマップされている場合はすぐに停止/一時停止します。その後、ファイルの復元機能を使用して、OneDrive または SharePoint ライブラリ全体を以前の時刻に復元できます。 

復元方法の詳細については、以下を参照してください。

- [OneDrive ライブラリを復元する](https://support.office.com/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a150)
- [ドキュメントライブラリを復元する](https://support.office.com/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a?ui=en-US&rs=en-US&ad=US)
