# <a name="microsoft-bookings-payment-options"></a>Microsoft Bookings 支払オプション

Microsoft Bookings 支払いオプションは 2020 年 3 月に廃止されました。 組織は現在、サードパーティのソリューションを使用して、予約に支払オプションを追加しています。
