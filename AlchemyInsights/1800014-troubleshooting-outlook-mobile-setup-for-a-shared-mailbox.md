# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>共有メールボックスに対して Outlook mobile のセットアップのトラブルシューティング

モバイル デバイスにインストールされている Outlook のアプリケーションからは、共有メールボックスにアクセスできません。この機能を作成しますか。Outlook のエンジニア リング チームを知ることができます。[投票は、ここ](https://go.microsoft.com/fwlink/?linked=862116)!
  
だけブラウザーを使用して共有されているメールボックス開くでモバイル デバイスで、Office 365 にサインインして、web 上で Outlook を移動できます。手順については、 [web 上の Outlook で共有されているメールボックスの追加](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)を参照してください。Web 上の Outlook から、共有メールボックスにアクセスすることになります。
  
詳細: [iOS および Android の Outlook のアプリケーションでヘルプを表示します。](https://support.office.com/en-us/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
