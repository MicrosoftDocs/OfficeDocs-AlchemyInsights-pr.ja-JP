# <a name="restore-a-user-in-office-365"></a>Office 365 でユーザーを復元する

次の手順を使用して Office 365 のユーザーを復元します。
  
1. [ユーザー\>ユーザーを削除する](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/deletedusers)です。
    
2. ユーザーを選択し、[**リストア**] を選択します。
    
3. 画面の指示に従ってパスワードを設定します。
    
4. **電子メールを送信し閉じる**をクリックしています。
    
簡単でしたか。詳細とスクリーン ショット付きの手順の詳細をチェック アウトこの記事: [Office 365 のユーザーを復元します。](https://support.office.com/article/https://support.office.com/en-us/article/Restore-a-user-in-Office-365-2c261e42-5dd1-48b0-845f-2a016d29cfc1.aspx)
