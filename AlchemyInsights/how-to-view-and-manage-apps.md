# <a name="how-to-view-and-manage-apps"></a>アプリを表示および管理する方法

Microsoft 365 管理センターでアプリを表示できます。 

1. 管理センターで、[**課金**]  >  [**[お使いの製品](https://go.microsoft.com/fwlink/p/?linkid=842054)**] ページに移動します。

2. [**[お使いの製品](https://go.microsoft.com/fwlink/p/?linkid=842054)**] ページで、[**アプリ**] タブを選択します。
