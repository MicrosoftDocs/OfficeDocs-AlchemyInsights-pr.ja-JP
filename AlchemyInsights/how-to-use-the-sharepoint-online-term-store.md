# <a name="how-to-use-the-sharepoint-online-term-store"></a>SharePoint のオンラインの用語ストアを使用する方法

用語ストアは、Office 365 のグローバル管理者、SharePoint 管理者、用語ストア管理者を使用し、クラシックの SharePoint Online 管理センターを参照しています。 
  
SharePoint のオンライン用語ストアと SharePoint Online 内の分類を使用する方法の詳細については、「[管理されたメタデータの概要](https://go.microsoft.com/fwlink/?linkid=2044674&amp;clcid=0x409)」を参照してください。
  
用語が表示されないようにしたり、断続的に表示されない問題が発生した場合は、24 時間、待機して、問題が解決しない場合は、サポート リクエストをログに記録し、ことをお勧めします。キャッシュの問題が存在する可能性があります。 または 24 時間できますが、長い時間いてもこの問題の解決策を既に動作可能性があります。
