# <a name="prepaid-renewal"></a>プリペイドの更新

1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. プリペイド サービスの**拡張**をクリックし、拡張するかを選択します。
    
3. **プロダクト キー**のオプションを選択します。(これが小売店で購入したプロダクト キーである場合があります他の支払方法を使用して拡張するためのオプション。)
    
4. プロダクト キーを入力します。
    
5. 情報が正しいかどうかを確認し、**次へ**を選択します。
    
6. 既存のライセンスの終了日を延長し、[**次**] を選択するオプション ボタンを選択します。
