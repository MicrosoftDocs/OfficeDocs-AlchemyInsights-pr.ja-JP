# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Sharepoint グループを使用して SharePoint サイトの権限をカスタマイズする方法 

SharePoint グループとは、すべてのユーザーがサイトとコンテンツに同じアクセス許可を持っているユーザーの集合です。 一度に1人のユーザーにアクセス許可を割り当てる代わりに、グループを使用して、複数のユーザーに同じアクセス許可レベルを一度に割り当てることができます。
