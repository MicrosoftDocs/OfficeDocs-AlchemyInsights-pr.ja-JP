# <a name="delete-a-sharepoint-page"></a>SharePoint ページを削除する

1. ページがあるサイトに移動します。
    
2. 左側にある [**サイト コンテンツ**] をクリックします。 
    
3. [サイト コンテンツ] ページの左側にある [**ページ**] をクリックします。 
    
4. ページを選択し、ページの右側にある**省略記号 (...)** をクリックします。 
    
5. [**削除**] をクリックします。
