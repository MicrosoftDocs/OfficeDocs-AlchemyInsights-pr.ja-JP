# <a name="create-and-use-a-shared-mailbox"></a>共有メールボックスを作成および使用する

共有メールボックスを使用すると、複数のユーザーは、追加のライセンスを割り当てることなく、1 つのエイリアス (**sales@contoso.com** など) で電子メールを送受信できます。
  
「[共有メールボックスを作成する](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)」では、次の方法ついて説明しています。 
  
- 共有メールボックスを作成する
    
- ユーザーに共有メールボックスを使用するためのアクセス許可を付与する
    
- Outlook または in Outlook on the web で共有メールボックスにアクセスする
