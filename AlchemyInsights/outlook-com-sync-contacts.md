# <a name="sync-contacts"></a>連絡先を同期する

Outlook では、[ユーザー](https://outlook.live.com/people/)が作成した連絡先を、Outlook for Android や Outlook for iOS など、他のデバイスやアプリ間で同期します。 [連絡先を作成](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01)する方法について説明します。

インポートされた連絡先 (たとえば、Outlook.com アカウントに接続した別の電子メールアカウントからの連絡先) は同期されません。 [連絡先をインポート](https://support.office.com/article/285a3b55-8d93-4ac8-93df-43fffd13b2f1)する方法について説明します。
