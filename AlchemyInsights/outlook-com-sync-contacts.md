# <a name="sync-contacts"></a>連絡先を同期する

Outlook では、[[連絡先]](https://outlook.live.com/people/) に作成した連絡先が別のデバイスとアプリ (Outlook for Android や Outlook for iOS など) のすべてに同期されます。 [連絡先の作成](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01)方法を確認してください。

インポートした連絡先 (たとえば、Outlook.com アカウントに接続した別の電子メール アカウントの連絡先) は同期されません。 [連絡先のインポート](https://support.office.com/article/285a3b55-8d93-4ac8-93df-43fffd13b2f1)方法を確認してください。
