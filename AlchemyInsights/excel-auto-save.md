# <a name="what-is-autosave"></a>自動保存とは

自動保存は、Excel、Word、および PowerPoint for Microsoft 365 サブスクライバーで使用可能な機能であり、作業中にファイルを自動的に保存します。 

よく寄せられる質問については、「[自動保存とは](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)」を参照してください。
