# <a name="cant-get-verification-codes"></a>確認コードを取得できません

連絡用メールまたは電話番号にアクセスできない場合は、Microsoft アカウントの[セキュリティの基本](https://account.microsoft.com/security)ページに移動し、[サインイン] ページで [**これらのユーザーがいません**] を選択し、指示に従ってセキュリティ情報を置き換えます。

*注:*[Microsoft Authenticator アプリ](https://go.microsoft.com/fwlink/?linkid=2016117)は、確認コードを提供するためにインターネット接続を必要としません。

詳細については、「 [Microsoft アカウントのセキュリティ情報と確認コード](https://support.microsoft.com/help/12428/)」を参照してください。
