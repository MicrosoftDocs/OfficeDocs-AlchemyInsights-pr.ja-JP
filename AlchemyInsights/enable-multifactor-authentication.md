# <a name="enable-multi-factor-authentication"></a>多要素認証を有効にします。

1. **ユーザー**に移動\>**アクティブなユーザー**
    
2. [**詳細**] をクリックして\>**セットアップ Azure の多要素認証**です。 
    
3. MFA を有効にするユーザーを検索します。すべてのユーザーを表示するのには、上部にある多要素認証の状態のビューを変更する必要があります。
    
4. ユーザーを有効にする] チェック ボックスをチェックします。
    
5.  適切なユーザー情報] ウィンドウで、[**クイック ステップ**が有効にするを参照してくださいしてユーザー設定を管理します。**有効にする**を選択します。 
    
6. ダイアログ ボックスを開くと、**多要素認証を有効にする] をクリックします**。
