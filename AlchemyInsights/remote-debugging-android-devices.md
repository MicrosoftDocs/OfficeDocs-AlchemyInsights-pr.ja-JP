# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>Microsoft Edge を使用して、開発用コンピューターから Android デバイスのリモートデバッグを行う

コンピューターから Android デバイス上のライブ コンテンツをリモートでデバッグするには、次の操作を行います。

1. リモートデバッグ用に Android デバイスをセットアップし、開発用コンピューターから検出します。
2. 開発用コンピューターから Android デバイスのライブ コンテンツを検査およびデバッグします。
3. Android デバイスのコンテンツを開発用コンピューターの DevTools インスタンスに Screencast します。
