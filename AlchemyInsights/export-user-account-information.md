# <a name="export-user-account-information"></a>ユーザーアカウント情報をエクスポートする

ライセンス、パスワードポリシー、ブロックされているアカウント、AAD 同期状態などの情報を含むユーザーアカウント情報をエクスポートするには、[**ユーザー** > の[アクティブなユーザー](https://go.microsoft.com/fwlink/p/?linkid=834822) ] ページに移動し、[**ユーザーのエクスポート**] を選択します。
