# <a name="export-user-account-information"></a>ユーザーアカウント情報をエクスポートする

Microsoft 365 管理センターでユーザーアカウント情報をエクスポートするには、ライセンス、パスワードポリシー、ブロックされているアカウント、AAD 同期状態などの情報を含め、[**ユーザー** > の[アクティブなユーザー](https://go.microsoft.com/fwlink/p/?linkid=834822) ] ページに移動して、[エクスポート] を選択します。 **ユーザー**。
