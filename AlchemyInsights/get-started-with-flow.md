# <a name="get-started-with-microsoft-flow"></a>マイクロソフトのフローを開始します。

マイクロソフトのフローを次の操作を実行できます。
  
- テンプレートおよびサービスを簡単に検索します。
    
- 最も人気のあるサービスを選択します。
    
- 各フローの概要を参照してください。
    
[マイクロソフトのフローの詳細を表示します](https://go.microsoft.com/fwlink/?linkid=874446)
