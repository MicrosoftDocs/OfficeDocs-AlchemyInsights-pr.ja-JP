# <a name="spam-filter-not-working"></a>スパム フィルターが機能していません

Outlook.com で受信したメッセージがフィッシング詐欺またはスプーフィング詐欺であると思われる場合は、メッセージを選択し、[**迷惑** \> **フィッシング**] を選択してフィッシング詐欺メールとしてマークします。 乱用、フィッシング、およびスプーフィングの詳細については、「」を[参照](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy)してください。 電子メールアドレスをブロックする、アドレスまたはドメインを差出人セーフリストに追加する、または他の迷惑メールオプションを適用するには、[**迷惑メールのオプション**] を選択します。
