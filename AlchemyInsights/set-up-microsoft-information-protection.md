# <a name="set-up-microsoft-information-protection"></a>Microsoft の情報保護を設定する

機密情報が確実に保護されるように、情報保護戦略に適用できる機能の概要を把握します。 機密情報を発見、分類、保護、および監視する 4 段階のライフサイクル アプローチを使用します。 「Microsoft の情報保護のセットアップ」ガイドには、これらの各段階を完了するためのガイダンスが記載されています。
