# <a name="policy-error-when-sharing-a-calendar"></a>予定表を共有する場合、ポリシー エラー

カレンダーを共有しようとしている場合は、エラーが発生する「ポリシーにより 1 つまたは複数の受信者には、このレベルでアクセス許可を付与する」は、[このトピック](https://support.microsoft.com/help/3187524/policy-does-not-allow-granting-permissions-at-this-level-to-one-or-mor)を参照してください。
