# <a name="add-seats-to-a-prepaid-subscription"></a>プリペイド サブスクリプションに座席を追加します。

1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. プリペイド サービスのユーザーを追加するかを選択します。
    
3. **その他のアクション**メニューで、選択**更新トークンを使用してライセンスを追加または**。
    
4. **新しいと未使用のプロダクト キー**のオプションを選択します。(小売店で購入したプロダクト キーを使用してこのサブスクリプションが作成されている場合するがあります他の支払方法でユーザーを追加するオプションです。) 
    
5. [ **次へ** ] を選びます。
    
6. プロダクト キーを入力し、**次へ**を選択します。
    
7. 多くのユーザーを追加、**次へ**を選択し、**償還**を選択するオプションを選択します。
