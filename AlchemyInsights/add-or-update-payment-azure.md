# <a name="update-billing-info-under-azure-enterprise-agreement"></a>Azure エンタープライズ契約に基づく請求情報の更新

Azure エンタープライズ サブスクリプションの支払い情報の変更はサポートされていません。

既存の請求書の住所を変更する場合は、Microsoft の担当者にお問い合わせください。
