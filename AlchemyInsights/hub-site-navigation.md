# <a name="sharepoint-hub-site-navigation"></a>SharePoint ハブサイトのナビゲーション

ハブサイトの上部、およびサイトナビゲーションの上にある関連付けられたすべてのサイトは、ハブサイトのナビゲーションバーです。 ハブサイトの所有者である場合は、[ナビゲーションバー](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav)をカスタマイズできます。 

**注:** コミュニケーションサイト、またはモダンテンプレートをハブサイトとして使用するチームサイトを選択することをお勧めします。 従来のチームサイトを使用している場合、ハブナビゲーションはモダンページにのみ表示され、ハブサイトの設定はモダンページにのみ表示されます。
