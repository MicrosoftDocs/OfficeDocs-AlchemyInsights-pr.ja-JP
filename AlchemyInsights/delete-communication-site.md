# <a name="delete-a-communication-site"></a>コミュニケーション サイトを削除する

1. サイトにアクセスします。
    
2. 右上の [設定] アイコンをクリックします。
    
3. [**サイトの設定**] をクリックします。[**サイト情報**] で [**このサイトを削除**] をクリックして、[**削除**] をクリックします。
