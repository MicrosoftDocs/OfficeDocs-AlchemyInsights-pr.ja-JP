# <a name="delete-a-communication-site"></a>コミュニケーション サイトを削除します。

1. サイトに移動します。
    
2. 右上の [設定] アイコンをクリックします。
    
3. [**サイトの設定**] をクリックします。[**サイト情報**]**このサイトを削除**] をクリックし、[**削除**] をクリックします。
