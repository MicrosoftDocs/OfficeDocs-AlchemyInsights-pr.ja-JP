# <a name="setting-up-a-hybrid-environment"></a>ハイブリッド環境を設定する

ユーザーのデータの格納場所がオンプレミスのサーバーまたは Office 365 のいずれであっても、ユーザーがシームレスに共同作業を行えるようにするために、ハイブリッド環境の設定が必要な場合があります。オプションの概要については、「[オンプレミス環境と office 365 の統合](https://docs.microsoft.com/office365/enterprise/office-365-integration)」を参照してください。
