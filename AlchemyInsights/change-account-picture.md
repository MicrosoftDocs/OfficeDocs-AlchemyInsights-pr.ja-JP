# <a name="change-account-picture"></a>アカウントの画像を変更する

ユーザー名とパスワードを使用して Windows 10 にサインインしている場合は、ローカルアカウントを使用しています。 ローカルアカウントの画像を変更するには、次のようにします。

1. [**開始** > **設定** > ********] [情報] に移動します。 > 
2. [**画像の作成**] で、[**参照**] を選択します。 または、デバイスにカメラがある場合は、[**カメラ**] を選択して、selfie に移動します。 
    注: Windows では、最後に使用した3つの画像が記憶されています。 [**情報**] ページで画像を選択して、その画像に切り替えます。

電子メールアドレスとパスワードを使用してサインインする場合は、Microsoft アカウントを使用しています。 Microsoft アカウントの画像を変更するには、次のようにします。

1. Microsoft アカウントを使用して、[**情報**] ページにサインインします。
2. [**画像の追加**] または [**画像の変更**] を選択し、指示に従います。

[Microsoft アカウントの設定と管理に関するその他のヘルプを取得する](https://support.microsoft.com/products/microsoft-account?category=manage-account)
