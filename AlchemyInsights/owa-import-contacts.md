# <a name="import-contacts"></a>連絡先をインポートする

連絡先を Outlook on the web にインポートする前に、使用している他のメール プロバイダーから連絡先を .csv ファイルとしてエクスポートする必要があります。 その後、次の指示に従います。

1. ページの下隅で [ユーザー] アイコンを選択し、[ユーザー] ページに移動します。
2. ツールバーで、**[管理]** > **[連絡先のインポート]** の順に選択します。
3. **[参照]** を選択し、CSV ファイルを選択して、**[開く]** を選択します。
4. **[インポート]** を選択します。

詳細については、「[Outlook に連絡先をインポートする](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web)」を参照してください。
