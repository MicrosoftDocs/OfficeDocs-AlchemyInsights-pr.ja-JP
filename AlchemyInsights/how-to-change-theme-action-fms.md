# <a name="how-to-change-your-theme"></a>テーマを変更する方法

Outlook.com または Outlook on the web でテーマを変更するには、**[設定]** に移動し、**[テーマ]** でテーマを選択します。使用可能なテーマをすべて見るには、**[テーマ]** の下の **[すべて表示]** を選択します。
