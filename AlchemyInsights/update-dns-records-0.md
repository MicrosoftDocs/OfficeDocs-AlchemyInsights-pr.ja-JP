# <a name="update-dns-records"></a>DNS レコードを更新する

- [Office 365 で DNS レコードを更新する](https://support.office.com/article/B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23)

- [Office 365 でカスタムの DNS レコードを追加または編集する](https://support.office.com/article/AF00A516-DD39-4EDA-AF3E-1EAF686C8DC9)
