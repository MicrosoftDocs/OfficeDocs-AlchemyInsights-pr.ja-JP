# <a name="update-dns-records"></a>DNS レコードを更新します。

DNS レコードを設定することは、カスタム ドメイン名を使用して Office 365 サブスクリプションを取得する必要があります。追加または、DNS レコードを更新するには、以下のリンクを使用します。
  
- [Office 365 で DNS レコードを更新します。](https://support.office.com/article/B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23)
    
- [カスタムの DNS レコードを追加または編集する](https://support.office.com/article/AF00A516-DD39-4EDA-AF3E-1EAF686C8DC9)
