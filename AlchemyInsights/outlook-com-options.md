# <a name="how-to-find-options-in-outlookcom-settings"></a>Outlook.com 設定のオプションを確認する方法

ページの上部にある歯車アイコンを選択して、[設定] を開きます。[設定] ウィンドウの下部にある [[**すべての Outlook 設定を表示**](https://outlook.live.com/mail/options/general/timeAndLanguage)] を選択します。

全般、メール、予定表、(該当する場合は) プレミアムの設定とオプションが表示されます。
