# <a name="the-open-with-explorer-button-is-missing"></a>エクスプ ローラーのボタンを使用してオープンされていません

SharePoint および OneDrive の最近の経験では、**エクスプ ローラーで開く**] ボタンが表示されません。ファイル エクスプ ローラーでドキュメント ライブラリを開き、Internet Explorer で参照し、[表示] メニューを選択して\>**ファイルのエクスプ ローラーで表示**します。**ファイル エクスプ ローラーで [表示**] オプションはマイクロソフトのエッジ、Google Chrome または Firefox で利用可能ではありません。 
  
> [!NOTE]
> エクスプ ローラーで開くは、Internet Explorer でのみサポートされます。代わりに、[新しい OneDrive の同期クライアントを使用して SharePoint ファイルを同期](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx)する[オン ・ デマンドのファイル](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx)が用意されています。ローカル記憶域を使用しなくても OneDrive 内のすべてのファイルにアクセスします。
