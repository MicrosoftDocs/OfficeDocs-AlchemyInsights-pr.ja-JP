# <a name="resolve-the-word-could-not-create-the-work-file-error-message"></a>"Word で作業ファイルを作成できませんでした" というエラー メッセージを解決する

このエラーを修正するには、「[Microsoft Word で文書を保存するときに、"Word で作業ファイルを作成できませんでした" というエラーメッセージ](https://docs.microsoft.com/office/troubleshoot/word/word-could-not-create-the-work-file)」を参照してください。

Word 文書を開くときに問題が発生する場合は、次の操作を行います。

- [Office の更新プログラムをインストールします](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5)。 最新の Office 更新プログラムの一覧については、[最新の更新プログラム](https://docs.microsoft.com/officeupdates/office-updates-msi)を参照してください。
- [Word の破損した文書のトラブルシューティング](https://docs.microsoft.com/office/troubleshoot/word/damaged-documents-in-word)。
- [Office アプリケーションを修復する](https://support.office.com/Article/Repair-an-Office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b)。
