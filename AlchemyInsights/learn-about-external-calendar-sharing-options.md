# <a name="learn-about-external-calendar-sharing-options"></a>外部予定表共有オプションについて

組織外のユーザーとの会議をスケジュールする必要がある場合、Office 365 では「外部ユーザー」が予定表を表示できるようにすることができます。この外部ユーザーとは、空き時間情報を確認する必要があるが、Office 365 環境のユーザー アカウントを持っていないユーザーのことです。 詳細については、「[予定表を外部ユーザーと共有する](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users)」を参照してください。
