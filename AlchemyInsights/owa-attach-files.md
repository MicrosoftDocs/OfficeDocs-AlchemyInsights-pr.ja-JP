# <a name="how-to-attach-files-to-an-email"></a>電子メールにファイルを添付する方法

1. メッセージまたは予定表イベントの下部で、クリップ アイコンを選択します。
1. コンピューター上のファイルを添付する場合は **[このコンピューターから選択]** を、オンラインのファイルを添付するには **[クラウドの場所から選択]** を選択します。

詳細については、「[Outlook on the web でファイルを添付する](https://support.office.com/article/48b8dca1-7a76-43ce-97d1-e1cf73893f55)」を参照してください。
