# <a name="whats-a-hub-site"></a>ハブ サイトとは何ですか。

ハブ サイトは、サイトを整理し、一般的なナビゲーションを提供してとブランド化します。グローバルまたは SharePoint の管理者はハブ サイトを作成し、ハブ サイトのサイトに関連付けることができるユーザーを指定することになります。詳細については、 [Office 365 で新しいハブ サイト](https://go.microsoft.com/fwlink/?linkid=869388)を参照してください。
