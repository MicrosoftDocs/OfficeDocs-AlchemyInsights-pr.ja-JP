# <a name="how-do-i-add-a-credit-card"></a>クレジット カードを追加する方法は?

1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. 新しいクレジット カードで支払うサブスクリプションを選んで、[ **お支払いの詳細の更新**] を選択します。
    
3. [**追加または支払方法の選択**] ドロップダウンを選択し、**新しいクレジット カード**または**新しい銀行口座**を選択します。
    
4. 新しいクレジット カードの情報を入力し、[ 送信] を選択します。
