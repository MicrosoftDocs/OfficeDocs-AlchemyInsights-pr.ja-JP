# <a name="outlookcom-updates"></a>Outlook.com の更新

**最新の更新**

- 新しい Outlook.com でメールをもっと活用しましょう。 検索機能が効率化され、これまで以上に探しているものを見つけることが簡単になりました。 ファイルや添付ファイルのプレビューなどの新機能も追加され、受信トレイから移動することなく写真やファイルを見ることができます。 絵文字や GIF をメッセージに追加する事ができるようになりました。 [新しい Outlook.com. の詳細について](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d)

**メールの更新**

- Outlook.com のメールボックスが更新されていない場合は、 [Outlook.com のメール同期の問題を修正する](https://support.office.com/article/d39e3341-8d79-4bf1-b3c7-ded602233642)を参照ください。
