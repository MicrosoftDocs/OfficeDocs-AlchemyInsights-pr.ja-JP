# <a name="how-to-change-your-theme"></a>テーマを変更する方法

Outlook.com または web 上の Outlook で、テーマを変更するには、**設定**に移動![設定](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png)、[**テーマ**] テーマを選択します。使用可能なすべてのテーマを表示するには、[**テーマ**] の**すべてのビュー**を選択します。
