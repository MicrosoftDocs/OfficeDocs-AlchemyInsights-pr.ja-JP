# <a name="find-the-page-that-shows-my-contacts"></a>連絡先を表示するページを検索する

メールを検索するのと同じ方法で、連絡先を検索できます。
 
拡大鏡の検索アイコンをタップして、画面の上部にあるボックスに連絡先検索要求を入力します。 (**Protip**: 拡大鏡をダブルタップして、検索ボックスにフォーカスを移動することができます。)結果は [連絡先] セクションに表示されます。
