# <a name="issues-while-migrating-data-to-sharepoint-online"></a>データを SharePoint Online に移行する際の問題

100TB データを SharePoint Online に移行しているように見えます。 次の手順に従って、できる限り早くお客様を支援する必要があります。 

1. [**新しいサービスリクエスト**] を選択してから、[**新しいサービスリクエスト**] を選択します。 
2. タイトルと説明は、 **100TB を介して SharePoint の移行**として残します。
3. チケットが送信されたら、次の情報で更新してください。 

    - 移行の推定サイズ。
    - 移行を開始および完了する予測値。
    - SharePoint Server、Box、GDrive、ファイル共有など、コンテンツの移行元の場所について説明します。
