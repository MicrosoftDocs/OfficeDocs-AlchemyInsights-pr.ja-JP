# <a name="restoring-a-deleted-team-site"></a>削除されたチーム サイトを復元する

チーム所有者が誤ってチームを削除してしまうことがありますが、削除したチームは復元できます。
  
Microsoft Teams で削除されたチームを復元するには、「[削除されたチームを復元する](https://blogs.technet.microsoft.com/skypehybridguy/2017/07/23/restoring-a-deleted-team-in-microsoft-teams/)」の手順に従います。
