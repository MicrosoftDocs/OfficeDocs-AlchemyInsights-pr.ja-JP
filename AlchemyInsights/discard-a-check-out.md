# <a name="discard-a-check-out-from-a-document-library"></a>ドキュメント ライブラリからチェック アウトを破棄します。

1. チェック アウトを破棄するファイルを右クリックします。
    
2. **詳細**] をクリックし、**チェック アウトの破棄**] をクリックします。 
    
3. ダイアログ ボックスで、[* * チェック アウト * * チェック ボックスをオフにします。 
    
チェックインとチェック アウトに関する詳細情報は、[上の質問については、チェック アウト、チェックイン、およびバージョン](https://go.microsoft.com/fwlink/?linkid=2018786)を参照してください。
