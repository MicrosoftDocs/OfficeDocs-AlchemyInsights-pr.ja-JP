# <a name="missing-filesfolders-in-sharepoint-online"></a>SharePoint Online のファイル/フォルダーが見つからない

OneDrive ファイルがマルウェアによって削除、上書き、破損、または感染した場合、Microsoft はユーザーに回復プロセスを支援するさまざまなオプションを提供します。

- [OneDrive で欠落、削除、または破損したアイテムを復元する方法](https://go.microsoft.com/fwlink/?linkid=2125166)
