# <a name="too-many-licenses"></a>多くのライセンスですか。

多くのライセンスの利用料金をしている場合は、これは、通常する必要がありますし、追加のライセンスを購入することが原因です。
  
 **購入したライセンスの数を減らす**
  
1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. 、左側の元となるするにはライセンスを削除するサブスクリプションを選択します。
    
3. [ **ライセンスの追加/削除**] を選択します。
    
4. 必要なユーザーの合計数を入力します。たとえば、25 のライセンスがある現在、5 を削除する場合は、このフィールドに「20」と入力します。
    
5. [ **Submit**] を選びます。
