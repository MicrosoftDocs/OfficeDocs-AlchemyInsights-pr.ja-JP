# <a name="public-folder-access-from-mobile-devices"></a>モバイル デバイスからパブリック フォルダーのアクセス

パブリック フォルダーは、モバイル デバイスからアクセス可能ではありません。
  
パブリック フォルダーと同様の機能があり、モバイル デバイスからアクセスする場合より良い代替の[Office 365 のグループについて理解する](https://support.office.com/en-us/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)を参照してください。
