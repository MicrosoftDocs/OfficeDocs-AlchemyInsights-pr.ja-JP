# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 のアクセスが拒否されました、テナントがしきい値を超えています

このメッセージは、Office 365 の組織で大量のスパムまたは一括メールが送信された場合に発生します。このブロックを削除するには、理解して、サポート担当者に原因を説明するし、基になる問題を修正する必要があります。
