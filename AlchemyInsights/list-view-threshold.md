# <a name="list-view-threshold"></a>リスト ビューのしきい値

SharePoint のリストとライブラリには、最大 3000 万のアイテムを含めることができますが、5,000 を超えるアイテムがある場合、それらを操作しようとしたときにリスト ビューのしきい値エラーが表示されることがあります。 このしきい値は、サービスのパフォーマンスを維持するために適用されます。 これは変更できません。 また、フォルダーの継承を共有または破棄しようとすると、このエラーが発生する場合があります。

このしきい値に達しないように、次の操作を行います。

- [SharePoint で大規模なリストとライブラリを管理する](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59)

- [共有/継承の中断のエラー](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance)
