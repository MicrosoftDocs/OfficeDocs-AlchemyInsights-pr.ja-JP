# <a name="create-mailboxes-that-dont-require-licenses"></a>ライセンスを必要としないメールボックスを作成します。

- 複数のユーザーのグループの電子メール アドレスを必要な場合は、[共有メールボックスを作成する](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9)を参照してください。
    
- 共同作業し、電子メールやファイルを共有する人々 のチームを許可するには、 [Office 365 のグループについての説明](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2)を参照してください。
