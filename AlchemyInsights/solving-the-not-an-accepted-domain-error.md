# <a name="got-a-parameter-cannot-be-found-error"></a>"パラメーターが見つかりません" エラーを受け取った場合

**"パラメーター名 ' EmailAddresses と一致するパラメーターが見つかりません"** というエラーメッセージが表示された場合は、テナントの設定を完了するために Microsoft 365 がもう一度長い時間がかかることを意味します。
  
セットアップ プロセスは完了に最大 4 時間を必要とします。セットアップ プロセスが完了するまでしばらく待ってから、もう一度試してみてください。
