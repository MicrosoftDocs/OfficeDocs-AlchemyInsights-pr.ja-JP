# <a name="change-your-language-date-and-time-format-or-time-zone"></a>言語、日付と時刻の形式、またはタイム ゾーンを変更する

1. ページの上部で、**[設定]** > **[Outlook のすべての設定を表示]** の順に選択します。
2. **[全般]** > **[言語と時刻]** を選択します。
3. 使用する言語を選択します。
4. **[保存]** を選択します。

詳細については、「[Outlook on the web でタイム ゾーンと言語を変更する](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273)」を参照してください。
