# <a name="troubleshooting-issues-with-calendar-sharing"></a>予定表の共有に関する問題のトラブルシューティング

- アクセス許可の問題を診断するには、「[Exchange リモート接続アナライザーの空き時間](https://testconnectivity.microsoft.com/Default.aspx?testId=freeBusy)」を使用してください。これにより、非常に具体的なエラー メッセージが表示されます。
- [Microsoft 365 の Microsoft サポート/回復アシスタント](https://diagnostics.office.com/)を使用して、予定表の他のさまざまな問題を診断することもできます。 
- ユーザーがデスクトップ版の Outlook を使用している場合、[Outlook 用の予定表チェック ツール](https://www.microsoft.com/download/details.aspx?id=28786)で潜在的な問題を解決できます。

追加の使い方のリソース:

- [他のユーザーと Outlook 予定表を共有する](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
- [Outlook on the web の自分の予定表を仕事のために共有する](https://support.office.com/article/7ecef8ae-139c-40d9-bae2-a23977ee58d5)
