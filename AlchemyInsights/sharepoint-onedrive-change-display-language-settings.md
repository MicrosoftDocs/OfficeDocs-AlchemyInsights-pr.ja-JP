# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>SharePoint または OneDrive の表示言語の設定を変更する 

SharePoint または OneDrive の既定の言語設定を変更することはできません。これらの設定は、Office 365 サブスクリプションの作成時に定義されています。 ただし、表示言語の設定を変更することはできます。

個人の設定を変更するには、「[個人の言語と地域の設定を変更](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)する」を参照してください。

サイトのユーザーインターフェイスを変更するには、「[サイトのユーザーインターフェイスで使用できるようにする言語を選択](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)する」を参照してください。

言語設定のトラブルシューティングの詳細については、「 [SharePoint Online サイトコレクションの既定の言語を変更することはできません](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)」を参照してください。
