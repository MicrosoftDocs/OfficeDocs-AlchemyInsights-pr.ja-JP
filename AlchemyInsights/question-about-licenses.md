# <a name="question-about-licenses"></a>ライセンスに関する質問します。

場合は、購入したライセンスの数を把握するか、サブスクリプションのライセンスの数が割り当てられました。
  
1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. 詳細を知りたいサブスクリプションを選択します。
    
3. サービス カードの左側にある、ライセンスに関する情報です。
    
  - **加入日**数を利用するライセンスの数です。 
    
  - **割り当て**数は、ユーザーに割り当てられているライセンスの数です。 
    
4. すぎるがある場合や、ライセンスを追加する場合は、多くの場合、購入量を調整するのには**ライセンスの追加と削除]** を選択します。
