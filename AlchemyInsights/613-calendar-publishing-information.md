# <a name="calendar-publishing-information"></a>予定表の情報の公開

- 管理者を有効にしたり、予定表の公開と組織の共有の予定表を無効にすることができます。詳細については、[人が情報を共有できるようにカレンダーの発行を有効に](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC)し、[外部ユーザーと予定表の共有](https://support.office.com/article/FB00DD4E-2D5F-4E8D-8FF4-94B2CF002BDD)を参照してください。
    
- Outlook ユーザーは、独自のカレンダーを共有できます。詳細については、[他のユーザーと Outlook の予定表の共有](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)を参照してください。
