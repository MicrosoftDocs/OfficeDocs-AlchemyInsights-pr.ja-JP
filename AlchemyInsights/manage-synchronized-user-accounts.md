# <a name="manage-synchronized-user-accounts"></a>同期されたユーザー アカウントを管理します。

通常、Office 365 の管理センターを使用するには追加し、ユーザー アカウントを変更します。ただし、ディレクトリ同期または SSO が有効になっている場合に使用することはできません。代わりに、 [Office 365 の PowerShell でのユーザー アカウントのプロパティを構成する](https://docs.microsoft.com/en-us/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell )」に従って、Windows PowerShell ユーザーを管理します。
