# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Outlook.com のリンクが Firefox で機能しない

Outlook.com のメール メッセージのリンクが、Firefox ブラウザーで表示すると機能しません。 Mozilla では、Firefox の[修正に向けて取り組んでいます](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409)が、それまでは次の方法を使用してください。
  
- [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) などの別の Web ブラウザーを使用する。
- 閲覧ウィンドウにメールを表示する。詳細については、[Outlook.com メールボックスの外観を変更する](https://support.office.com/article/b41c2ecb-f23c-42b3-b7f8-659646d5e58c?wt.mc_id=Office_Outlook_com_Alchemy)方法を参照してください。
