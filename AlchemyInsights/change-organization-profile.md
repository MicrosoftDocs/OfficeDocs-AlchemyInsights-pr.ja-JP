# <a name="change-organization-profile"></a>組織プロファイルの変更

[[組織プロファイル](https://go.microsoft.com/fwlink/p/?linkid=2067339)の**設定** > ] ページの管理センターで、会社名、住所、技術連絡先情報などを変更します。
変更を保存できるように、* でマークされたすべての必須フィールドに入力してください。

[住所または請求先住所の変更について詳しくは、こちらを参照してください](https://docs.microsoft.com/en-us/office365/admin/manage/change-address-contact-and-more)。
