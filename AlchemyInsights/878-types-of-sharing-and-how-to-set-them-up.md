# <a name="types-of-calendar-sharing"></a>予定表の共有の種類

次は、予定表の共有の使用可能な種類とそれらを構成する方法があります。
  
- [共有メールボックスを作成する](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9.aspx)
    
- [会議室メールボックスと備品用メールボックス](https://support.office.com/article/9f518a6d-1e2c-4d44-93f3-e19013a1552b.aspx)
    
- [予定表グループを作成します。](https://support.office.com/article/8385667b-d758-4489-a53f-f542dd01e6ff.aspx)
    
- [Outlook の予定表を他のユーザーと共有します。](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88.aspx)
    
その他の予定表で予定表の空き時間情報が表示されない場合、または予定表の空き時間情報を他の人が表示されない場合[マイクロソフトのサポート、および Office 365 の回復時のアシスタント](https://diagnostics.office.com/)を使用します。
