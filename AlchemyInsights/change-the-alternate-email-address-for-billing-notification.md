# <a name="change-the-alternate-email-address-for-billing-notification"></a>請求通知の送り先である連絡用電子メール アドレスを変更する

課金通知の送り先である連絡用電子メール アドレスの変更次の手順を実行してください。
  
1. 管理センターで、[請求の通知](https://go.microsoft.com/fwlink/p/?linkid=853212)] ページに移動または**請求先**を選択して\>**の通知を請求**します。
    
2. 一覧から名前を選びます。
    
3. [ **連絡用メール アドレス**] ボックスに、連絡用メール アドレスを入力します。 
    
4. [ **保存**] を選択したら、完了です。 
    
スクリーン ショット付きの詳細と手順については、「[一般法人向け Office 365 の請求先住所を変更する](https://support.office.com/en-us/article/Change-your-billing-addresses-for-Office-365-for-business-a25c10d6-c1e9-4299-9185-25178df9eba6)」を参照してください。
