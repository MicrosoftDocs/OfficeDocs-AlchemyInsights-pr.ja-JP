# <a name="how-do-i-turn-off-skype-in-outlookcom"></a>Outlook.com で Skype を無効にするにはどうすればよいですか?

Skype を Outlook.com から削除することはできませんが、通知を無効にすることができます。

1. **設定** > の**一般的な** > **[通知](https://outlook.live.com/mail/options/general/notifications)** を開きます。 
2. [**チャットと呼び出し**] で、不要な通知の種類を無効にします。

Skype の使用の詳細については、Outlook.com の「 [skype と友人に問い合わせる」を](https://support.office.com/article/83c6a5b1-3921-479c-b9e9-e753ce59c1fa)参照してください。
