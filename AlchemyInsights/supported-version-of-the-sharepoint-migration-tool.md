# <a name="supported-version-of-the-sharepoint-migration-tool"></a>サポートされている SharePoint 移行ツールのバージョン



現在、SharePoint 移行ツールは、SharePoint 2013 からの移行のみをサポートしています。現時点では、バージョン 2010 および 2016 はサポートされていません。
  
詳細については、「[SharePoint 移行ツールの概要](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409)」を参照してください。
