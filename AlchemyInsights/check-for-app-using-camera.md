# <a name="check-for-app-using-camera"></a>カメラを使用してアプリを確認する

1. **[スタート]** を選択してから、**[設定]、[プライバシー]、[カメラ]** (ms-settings:privacy-webcam) の順に選択します。
2. アプリケーション リストで、「現在使用中」のアプリを見つけて、アプリケーションを閉じます。
