# <a name="changing-public-folder-permissions"></a>パブリック フォルダーのアクセス許可を変更します。

ユーザーは、Outlook でパブリック フォルダーのアクセス許可を変更できます。参照してください[を設定またはパブリック フォルダーのアクセス許可を変更する](https://support.office.com/en-us/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005)です。管理者は、Exchange 管理センターでのアクセス許可を変更することも。
  
- Office 365 の管理センターで**管理センター**に移動する\>**交換**します。
    
- **パブリック フォルダー**を選択します。
    
そこからアクセス許可をセキュリティ グループのユーザーを割り当てることで、個々 のパブリック フォルダーのアクセス許可を変更できます。パブリック フォルダーのアクセス許可を変更するには、それの所有者を指定する必要があります。
