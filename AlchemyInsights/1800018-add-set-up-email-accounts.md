# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>Outlook で Exchange 電子メール アカウントの設定の問題

、Outlook で Exchange アカウントの設定の問題が発生した場合は、影響を受けるコンピューターの[Outlook アカウントのセットアップ ツール](https://aka.ms/SaRA-OutlookSetupProfile)を実行できます。さまざまなチェックが実行され、アカウントのセットアップの問題の解決策の候補を提供します。 
  
問題がある場合でも、 [Outlook を Office 365 での Exchange Online のメールボックスの Exchange の自動検出を使用して新しいプロファイルを設定できません](https://support.microsoft.com/en-us/help/2404385/outlook-can-t-set-up-a-new-profile-by-using-exchange-autodiscover-for)が表示されます。
