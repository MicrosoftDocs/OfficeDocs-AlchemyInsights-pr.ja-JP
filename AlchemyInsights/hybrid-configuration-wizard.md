# <a name="hybrid-configuration-wizard"></a>ハイブリッド構成ウィザード

ハイブリッド構成ウィザードでは、オンプレミスの Exchange 組織と Office 365 の間で新しいハイブリッド展開をセットアップするために役立つアプリケーションです。既にハイブリッド展開をセットアップする場合、ウィザードでは、ハイブリッド展開の設定を更新します。詳細については、チェック アウトします。
  
- [ハイブリッド構成ウィザードを実行します。](https://technet.microsoft.com/en-us/library/mt595788%28v=exchg.150%29.aspx)
    
- [ハイブリッド構成ウィザード](https://technet.microsoft.com/en-us/library/hh529921%28v=exchg.150%29.aspx)
