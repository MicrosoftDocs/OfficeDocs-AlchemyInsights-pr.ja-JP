# <a name="long-running-script"></a>長時間実行されるスクリプト

Outlook.com における長時間実行されるスクリプトの問題が報告されています。この問題を回避するには、Outlook.com のベータ版を使用してください。
  
[Outlook.com のベータ版に関する詳細をご覧ください。](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)
