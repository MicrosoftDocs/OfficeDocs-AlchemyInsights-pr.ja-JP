# <a name="delete-a-sharepoint-site"></a>SharePoint サイトを削除する

1. 新しい SharePoint 管理センターの[サイト] ページで、削除するサイトの横にあるチェック ボックスをクリックします。
    
2. [リボン] ツールバーで、**[削除]** をクリックします。
