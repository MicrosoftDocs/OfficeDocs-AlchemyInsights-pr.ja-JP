# <a name="delete-a-sharepoint-site-collection"></a>SharePoint のサイト コレクションを削除する

1. 従来の SharePoint 管理センターの [サイト コレクション] ページで、サイト コレクションの隣のチェック ボックスを選択します。
    
2. [サイト コレクション] タブで **[削除]** を選択します。
