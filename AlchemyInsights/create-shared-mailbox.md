# <a name="create-a-shared-mailbox"></a>共有メールボックスを作成する

共有メールボックスを使用すると、複数のユーザーは、追加のライセンスを割り当てることなく、1 つのエイリアス (sales@contoso.com など) で電子メールを送受信できます。共有メール ボックスを作成するには、[共有メール ガイド](https://portal.office.com/adminportal/home)を参照してください。
  
共有メール ボックスの使用に関するヒントと詳細については、記事「[共有メールボックスを作成する](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)」を参照してください。
