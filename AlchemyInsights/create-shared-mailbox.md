# <a name="create-a-shared-mailbox"></a>共有メールボックスを作成する

共有メールボックスでは、複数のユーザー別のライセンスを割り当てることのない sales@contoso.com と同様に、エイリアスに電子メールの送受信を行うことができます。共有メールボックスを作成するには、[共有の電子メールのガイド](https://portal.office.com/adminportal/home)を移動します。
  
ヒントと共有されているメールボックスの使用に関する詳細な情報については、[共有メールボックスを作成する](https://support.office.com/en-us/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)資料を参照してください。
