# <a name="create-a-shared-mailbox"></a>共有メールボックスを作成する

共有メールボックスを使用すると、複数のユーザーは、追加のライセンスを割り当てることなく、1 つのエイリアス (sales@contoso.com など) で電子メールを送受信できます。 

共有メール ボックスの使用に関するヒントと詳細については、記事「[共有メールボックスを作成する](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)」を参照してください。
