# <a name="block-email-forwarding"></a>メールの転送をブロックします。

電子メールを特定のメールボックスに転送を無効にするには、[メールの転送設定](https://support.office.com/client/15abf81d-5c5d-49da-ac81-1b4daa1809f6)を参照してください。
  
[自動電子メールが Exchange オンラインの転送を禁止するのには多くの方法](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/)、組織全体の電子メールの転送をブロックします。
