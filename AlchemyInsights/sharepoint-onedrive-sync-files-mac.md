# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>OneDrive または SharePoint のファイルをインストール、セットアップ、および同期する (Mac) 

Mac または mac [OS X 上の onedrive 同期クライアントによるファイルの同期](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f)に関する指示に従って、onedrive または SharePoint ファイルを mac 上にインストール、セットアップ、および同期します。

OneDrive アプリがインストールされていて、Mac にセットアップされているが、ユーザーが同期の問題が発生している場合は、「 [onedrive との同期の問題を修正する (mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US)」を参照してください。

> [!NOTE]
> Windows 用 OneDrive sync の既知のフォルダーの移動機能は、Mac では使用できません。
