# <a name="restore-a-deleted-sharepoint-list-or-library"></a>削除した SharePoint リストまたはライブラリを復元する

1. SharePoint サイトに移動します。
    
2. 左側のウィンドウにある **[ごみ箱]** をクリックします。[ごみ箱] が表示されていない場合は、[設定] アイコンをクリックして、**[サイト コンテンツ]** をクリックします。[ごみ箱] は最上部のコマンド バーの右端にあります。
    
3. 復元するアイテムの左側をクリックします。
    
4. **[復元]** をクリックします。
    
削除したアイテムは 93 日間保持されます。対象のアイテムが見つからない場合は、このごみ箱から削除され、サイト コレクションのごみ箱に移動されている可能性があります。ごみ箱の最下部にある **[第 2 段階のごみ箱]** リンクをクリックすると確認できます。
