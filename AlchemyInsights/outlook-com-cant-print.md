# <a name="unable-to-print"></a>印刷できない

一部のユーザーが Safari ブラウザーや Edge ブラウザーを使用するときに印刷できないという問題が生じていることを把握しています。 この問題が解決するまでの間、別のブラウザーをご使用ください。 ご不便をおかけして申し訳ございません。
