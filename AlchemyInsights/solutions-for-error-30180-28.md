# <a name="solutions-for-error-30180-28"></a>エラー 30180-28 の解決方法

低速なインターネット接続では、Office の更新時に[エラー 30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) が発生することがあります。この問題を修正するために、次のことを試してみてください。 
  
- コンピューターを再起動してから [Office を更新する](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA)。
    
- 上記の手順を実行しますが、Office の更新中は有線接続を使用します。
