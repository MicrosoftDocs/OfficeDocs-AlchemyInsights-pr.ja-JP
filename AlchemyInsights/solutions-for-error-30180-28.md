# <a name="solutions-for-error-30180-28"></a>30180-28 のエラーのためのソリューション

低速なインターネット接続がある場合は、Office 更新プログラムの実行中に[エラー 30180-28](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA)を見ること。ここでは、いくつかの点が問題を解決しようとすることができます。 
  
- コンピューターを再起動し、 [Office を更新](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA)します。
    
- 上記の手順に従いますが、Office を更新する際にワイヤード (有線) 接続を使用します。
