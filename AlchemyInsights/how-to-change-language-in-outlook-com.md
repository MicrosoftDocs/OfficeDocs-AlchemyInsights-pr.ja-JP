# <a name="change-your-language"></a>言語を変更する

1. 右上隅の **[設定]** \> **[オプション]** を選択します。
    
2. **[オプション]** で、**[全般]**、**[地域とタイム ゾーン]** の順に選択します。
    
3. **[地域とタイム ゾーンの設定]** で、言語を選択します。
    
4. **[保存]** を選択します。
    
詳細については、[Outlook.com でのタイム ゾーンおよび言語の設定の変更](https://go.microsoft.com/fwlink/p/?linkid=873132)に関するページを参照してください。
