# <a name="change-your-language"></a>言語を変更する

1. 右上隅で、**[設定]** \> **[Outlook のすべての設定を表示]** の順に選択します。
    
2. **[全般]** から [**[言語と時刻]**](https://outlook.live.com/mail/options/general/timeAndLanguage) の順に選びます。
    
3. お使いの言語を選択します。
    
4. [**保存**] を選択します。
    
詳細については、[Outlook.com でのタイム ゾーンおよび言語の設定の変更](https://go.microsoft.com/fwlink/p/?linkid=873132)に関するページを参照してください。
