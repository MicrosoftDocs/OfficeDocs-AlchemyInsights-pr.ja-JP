# <a name="enable-versioning-for-a-list-or-library"></a>リストまたはライブラリのバージョン管理を有効にする

1. リストまたはライブラリに移動します。
    
2. [**設定**] をクリックし、[**ライブラリの設定**] または [**リストの設定**] をクリックします。
    
3. [**バージョン管理の設定**] をクリックします。
    
4. 目的の設定を選択し、[**OK**] をクリックします。
