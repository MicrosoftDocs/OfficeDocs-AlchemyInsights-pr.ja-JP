# <a name="enable-versioning-for-a-list-or-library"></a>リストまたはライブラリのバージョン管理を有効にします。

1. リストまたはライブラリに移動します。
    
2. **の設定**] をクリックし、**ライブラリの設定**] または [**リストの設定**] をクリックします。
    
3. **バージョン管理設定**をクリックします。
    
4. 目的の設定を選択し、し、[ **OK**] をクリックします。
