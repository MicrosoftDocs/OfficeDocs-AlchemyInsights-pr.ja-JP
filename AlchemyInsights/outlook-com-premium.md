# <a name="outlookcom-premium"></a>Outlook.com プレミアム

申し訳ございません、Outlook.com プレミアムは受付を終了しました。 多くの [機能とメリット](https://support.office.com/article/78c6089c-7faf-44f5-82e2-efa9ebb921d2) は [Office 365 Home と Office 365 Personal のサブスクリプション契約](https://go.microsoft.com/fwlink/?linkid=2017122)に含まれています。

Outlook.com プレミアム サブスクリプションをすでに持っている場合は、[Outlook.com プレミアム よく寄せられる質問](https://support.office.com/article/cd5f03f6-1407-456a-9410-f8f24804746b)を参照ください。
