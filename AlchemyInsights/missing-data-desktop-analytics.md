# <a name="cant-create-a-compatibility-assessment"></a>互換性評価を作成できません

Configuration Manager はデバイスを正常に構成しましたが、Desktop Analytics は互換性評価を作成できません。 デバイスの構成 (センサス) またはインストール済みのアプリ (インベントリ) の完全なデータセットがありません。

この問題は多くの場合デバイスの再試行時に自動的に修正されます。 問題が解決しない場合は、デバイスがサービスと通信できることを確認してください。 詳細については、[エンドポイント](https://docs.microsoft.com/sccm/desktop-analytics/enable-data-sharing#endpoints)と [Missing Data (不足しているデータ)](https://docs.microsoft.com/sccm/desktop-analytics/monitor-connection-health#missing-data) を参照してください。
