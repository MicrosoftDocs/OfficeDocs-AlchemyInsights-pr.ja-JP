# <a name="print-results-in-a-summary-of-responses"></a>応答の概要の結果を印刷する

[Microsoft Forms の結果](https://support.office.com/article/print-a-form-22100b98-ba3c-41c1-9513-f76caca664fc)は、回答の要約として、または個別に印刷することができます。 

返信なしでフォームを印刷することはできませんが、 [Microsoft Forms UserVoice サイト](https://microsoftforms.uservoice.com/forums/386451-welcome-to-microsoft-forms-suggestion-box)にアクセスして、この機能を要求または投票することができます。
