# <a name="help-enabling-school-data-sync"></a>学校データ同期を有効にするためのサポートを受ける

学校データ同期を有効にするためのサポートが必要ですか? Microsoft 365 FastTrack プログラムでサポートを受けられる資格をお持ちであることをご存じでしたか?

[FastTrack Center](https://www.microsoft.com/fasttrack) にアクセスして、お客様の組織の学校データ同期を有効にするためのサポート要求をお送りください。
