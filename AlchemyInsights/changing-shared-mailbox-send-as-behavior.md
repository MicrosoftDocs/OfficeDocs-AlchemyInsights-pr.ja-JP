# <a name="changing-shared-mailbox-send-as-behavior"></a>共有メールボックスとして送信する動作を変更します。

既定では、共有メールボックスから送信されたメッセージは、共有メールボックスの [送信済みアイテム] フォルダーには保存されません。代わりに、メッセージを送信したユーザーの [送信済みアイテム] フォルダーに保存されます。
  
Office 365 管理センターで、送信された電子メールを表示するすべてのメンバーにする場合、共有されているメールボックスの詳細を編集し、**送信済みアイテム**を選択して\>**を編集**します。
  
詳細については、[共有メールボックスを作成する](https://support.office.com/en-us/article/create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)を参照してください。
