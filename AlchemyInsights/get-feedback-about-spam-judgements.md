# <a name="new-feature-get-feedback-about-spam-judgments"></a>新機能: スパムの判断に関するフィードバックを取得する

管理者は、不審な電子メールメッセージ、Url、およびコンテンツを、投稿された[エクスプローラー](https://protection.office.com/reportsubmission)で Microsoft に簡単に報告できるようになりました。 これを使用して、正当なメッセージが誤ってフラグが設定された理由を理解することもできます。 詳細については、この[アナウンス](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622)を参照してください。

これを easer にするために、ユーザーが[レポートメッセージアドイン](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview)を使用して直接 Microsoft に報告できるようにすることができます。
