# <a name="get-feedback-about-spam-judgments"></a>スパムの判断に関するフィードバックを取得する

管理者は、疑わしい電子メールメッセージ、Url、およびコンテンツを、投稿された[エクスプローラー](https://protection.office.com/reportsubmission)で、Microsoft に簡単に報告できます。 これを使用して、正当なメッセージに誤ってフラグが付けられた理由を理解することもできます。 詳細については、この[お知らせ](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622)を参照してください。

簡単にするために、ユーザーが[レポート メッセージ アドイン](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview)を介して Microsoft に直接報告できるようにすることができます。
