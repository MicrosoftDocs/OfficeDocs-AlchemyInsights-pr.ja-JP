# <a name="how-to-type-special-characters-on-a-mac"></a>Mac で特殊文字を入力する方法

Mac で特殊文字を入力するには、キーボード ショートカット ⌘ **[Command]** + **[Control]** + **[Space]** を使って、**[特殊文字]** メニューを開きます。

また、@ のクイック ショートカットは **[ALT]** + **64** です。
