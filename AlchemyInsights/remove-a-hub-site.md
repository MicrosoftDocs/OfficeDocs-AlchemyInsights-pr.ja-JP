# <a name="remove-a-hub-site"></a>ハブ サイトを削除する

新しい SharePoint 管理センターでハブ サイトの登録を解除する
  
1. 新しい SharePoint 管理センターの **[サイト]** で、**[アクティブなサイト]** をクリックします。 
    
2. サイトを選択して、**[ハブ サイト]** をクリックしてから **[バブ サイトとしての登録を解除する]** をクリックします。 
    
3. **[OK]** をクリックします。 
    
[ハブサイトの詳細](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
