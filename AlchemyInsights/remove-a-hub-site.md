# <a name="remove-a-hub-site"></a>ハブ サイトを削除します。

新しい SharePoint 管理センターのハブ サイトの登録を解除します。
  
1. 新しい SharePoint 管理センターで、[**サイト**を**アクティブなサイト**をクリックします。 
    
2. サイトを選択して、**ハブ サイト**を**ハブ サイトとしての登録を解除**] をクリックします。 
    
3. **[OK]** をクリックします。 
    
[ハブ サイトの詳細を表示します](https://support.office.com/en-us/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
