# <a name="delete-the-office-document-cache"></a>Office ドキュメントキャッシュを削除する

**新しいバージョンのファイルが使用できる**ことを示す Excel プロンプトが表示された場合は、「 [office ドキュメントキャッシュを削除](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d)する」の手順に従って office ドキュメントキャッシュを削除してみてください。 その後、ファイルを再度開きます。
