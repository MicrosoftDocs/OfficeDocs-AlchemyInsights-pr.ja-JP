# <a name="delete-the-office-document-cache"></a>Office ドキュメント キャッシュを削除する

**a newer version of the file is available (新しいバージョンのファイルが利用可能です)** という Excel プロンプトを受け取った場合は、[Office ドキュメント キャッシュを削除する](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d)の指示に従って Office ドキュメント キャッシュを削除してみてください。 その後、ファイルを再度開いてみてください。
