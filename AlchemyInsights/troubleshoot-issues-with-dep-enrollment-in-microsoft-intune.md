# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>Microsoft Intune で DEP の登録に関する問題をトラブルシューティングします。

これで問題を解決するのには下記のリソースを確認します。 
  
1. DEP のデバイスを登録することができませんし、MFA (多要素認証) が有効にする場合は、MFA を無効にしてください。現在 MFA は、DEP の登録はサポートされていません
    
2. [Intune トラブルシューティングのポータル](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade)を使用して、診断し、一般的な登録の失敗を解決します。詳細については[このドキュメント](https://docs.microsoft.com/en-us/intune/help-desk-operators)を確認します。 
    
3. 各登録と解決策を防ぐための一般的なエラーの一覧については、これらのドキュメントを確認する:[トラブルシューティング ガイド](https://support.microsoft.com/en-us/help/4039809/troubleshooting-ios-device-enrollment-in-intune)と[トラブルシューティングのドキュメント](https://docs.microsoft.com/en-us/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)
    
4. [デバイス登録プログラムについて学習](https://docs.microsoft.com/en-us/intune/device-enrollment-program-enroll-ios)します。
