# <a name="admin-roles"></a>管理者権限

- 管理者役割の割り当てまたは詳細については、管理センターで [[役割](https://admin.microsoft.com/Adminportal/Home#/roles)] を選択してください。

- 次に、役割名を選択して [**役割の割り当て**] をクリックするか、[**役割の比較**] をクリックして詳細を確認します。
