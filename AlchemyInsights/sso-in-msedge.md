# <a name="single-sign-on-sso-in-microsoft-edge"></a>Microsoft Edge でのシングル サインオン (SSO)

Microsoft Edge では、次の種類のシングル サインオン (SSO) をサポートしています。
- PRT ベースの SSO
- シームレスな SSO
- WIA ベースの SSO
