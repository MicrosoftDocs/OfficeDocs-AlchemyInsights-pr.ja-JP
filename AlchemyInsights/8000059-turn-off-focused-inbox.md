# <a name="turn-off-focused-inbox"></a>優先受信トレイを無効にする

優先受信トレイを使用すると、最も重要なメールを優先的に扱うことができます。読む可能性の高いメールは [優先] 下に表示され、残りのメールは [その他] に移動されます。その仕組みについては、[こちらをご覧ください](https://support.office.com/article/f445ad7f-02f4-4294-a82e-71d8964e3978#bkmk_consumer)。優先受信トレイを使用しない場合は、下のボタンを選択します。
