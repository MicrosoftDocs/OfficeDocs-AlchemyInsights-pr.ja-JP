# <a name="turn-off-focused-inbox"></a>中心の受信トレイをオフにします。

フォーカスのある受信トレイのことが最も重要な要素に重点を置きます。電子メールをしているか、読み取ることがより詳細の下に表示され、他の残りの部分を移動します。については、そのしくみ、[これを読みます](https://support.office.com/article/f445ad7f-02f4-4294-a82e-71d8964e3978#bkmk_consumer)。中心の受信トレイを使用しない場合は、下のボタンを選択します。
