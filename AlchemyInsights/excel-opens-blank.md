# <a name="excel-opens-blank"></a>Excel が空白

Excel が起動しても、ファイルが表示されるはずの場所に空白の画面が表示される場合は、[ファイルアイコンまたはファイル名をダブルクリックしたときに excel を開くのトラブルシューティング](https://docs.microsoft.com/office/troubleshoot/excel/excel-opens-blank)を参照してください。
