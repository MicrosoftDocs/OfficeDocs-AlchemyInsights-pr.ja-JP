# <a name="convert-a-user-mailbox-to-a-room-or-equipment-mailbox"></a>ユーザー メールボックスを会議室や備品用メールボックスに変換します

あるメールボックスの種類を別の種類に変換することもできますが、これらのタスクを Exchange Online PowerShell を使用して実行する必要があります。詳細については、「[メールボックスを変換する](https://go.microsoft.com/fwlink/p/?LinkId=832875)」を参照してください。
