# <a name="unable-to-attach-or-upload-files-from-facebook"></a>Facebook からファイルを添付またはアップロードできません

Facebook の使用条件の変更により、2019年10月23日以降、添付ファイルを Facebook から直接アップロードすることができなくなり、Facebook がクラウドの場所の一覧に表示されなくなりました。 

[ストレージアカウント](https://go.microsoft.com/fwlink/?linkid=2111075)の他のクラウドの場所の一覧を管理することができます。

[ストレージアカウント](https://support.office.com/article/477cb7cc-5732-4c40-8f23-30472de8138a)の詳細について説明します。
