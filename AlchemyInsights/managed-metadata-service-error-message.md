# <a name="managed-metadata-service-error-message"></a>マネージ メタデータ サービスのエラー メッセージ

用語ストアを開くとエラー メッセージが表示ことがあります:"管理されたメタデータ サービスまたは接続は現在使用できません。メタデータ Web サービスの管理、アプリケーション プールが起動されていません。管理者に問い合わせてください。」
  
この問題を最初に見たから 24 時間以上が経過、サポート案件を記録してください。弊社は、24 時間が長い時間のように思われる可能性があることを理解します。多くの場合、既に取り組んでいますソリューションのため、少なくとも 24 時間を指定してください。
