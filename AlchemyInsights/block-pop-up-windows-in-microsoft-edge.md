# <a name="block-pop-up-windows-in-microsoft-edge"></a>Microsoft Edge でポップアップ ウィンドウをブロックする

1. **[... (設定など)]** > **[設定]** > **[サイトのアクセス許可]** の順に移動します。
2. **[ポップアップとリダイレクト]** を選択します。
3. **[ブロック]** をオンにします。
