# <a name="excel-autofilter-issue"></a>Excel のオート フィルターの問題

オート フィルター ドロップダウンが表示されないすべての利用可能なチェック ボックス、またはチェック ボックスがクリックできない場合がある場合は、Excel の 2016年を使用して、Excel の以前のビルドでの既知の問題である可能性があります。 
  
**バージョン 1707 (8326.2073)** 以降では、この問題は固定されます。更新プログラムをすぐに取得するには、Excel を起動し、**ファイル**を選択して\>**アカウント** \> **の更新オプション** \> **[今すぐ更新**します。
