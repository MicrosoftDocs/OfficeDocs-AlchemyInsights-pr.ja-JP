# <a name="error-onedrive-cannot-connect-to-windows"></a>エラー: OneDrive を Windows に接続できない

このエラー メッセージを解決するには、cldflt.sys ファイルを検索し、それが実行されているかどうかを確認します。実行されていない場合は、有効にしてください。 
  
cldflt.sys ファイルを有効にする方法については、「[OneDrive でファイルにアクセスすると、"OneDrive は Windows に接続できません" というエラーが発生する](https://go.microsoft.com/fwlink/?Linkid=2031032)」を参照してください。
