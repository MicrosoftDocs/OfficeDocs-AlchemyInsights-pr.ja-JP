# <a name="error-onedrive-cannot-connect-to-windows"></a>エラー: OneDrive は、Windows に接続できません。

エラー メッセージを解決するには、実行されている場合を参照してくださいしていない場合は、有効にする cldflt.sys ファイルを探します。 
  
Cldflt.sys ファイルを有効にする方法の詳細についてを参照してください[OneDrive は、Windows に接続できません"OneDrive 内のファイルにアクセスするとエラーが発生します](https://go.microsoft.com/fwlink/?Linkid=2031032)。
