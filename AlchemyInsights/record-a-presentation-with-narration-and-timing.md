# <a name="record-a-presentation-with-narration-and-timing"></a>プレゼンテーションをナレーションとタイミングとともに記録する

- **スライド ショーを記録する**: スライド ショーを記録するには、「[スライド ショーをナレーションとスライド切り替えのタイミングとともに記録する](https://support.office.com/article/Record-a-slide-show-with-narration-and-slide-timings-0B9502C6-5F6C-40AE-B1E7-E47D8741161C)」の手順を実行します。また、「[記録に関するヒント](https://support.office.com/article/Record-a-slide-show-with-narration-and-slide-timings-0B9502C6-5F6C-40AE-B1E7-E47D8741161C#OfficeVersion=Web)」も参照できます。
**注**: PowerPoint for the web ではスライド ショーの記録はサポートされていません。 
- **プレゼンテーションの記録と配布**について詳しくは、「[プレゼンテーションをビデオに変換する](https://support.office.com/article/Turn-your-presentation-into-a-video-C140551F-CB37-4818-B5D4-3E30815C3E83)」をご覧ください。
- **プレゼンテーションにビデオ ファイルを挿入する**: [コンピューターからビデオ ファイル](https://support.office.com/article/insert-and-play-a-video-file-from-your-computer-f3fcbd3e-5f86-4320-8aea-31bff480ed02)を挿入するか、[YouTube からビデオ](https://support.office.com/article/Insert-a-video-from-YouTube-or-another-site-8340ec69-4cee-4fe1-ab96-4849154bc6db)を挿入します。  [PowerPoint で選択したファイルからビデオを挿入できない](https://support.office.com/article/PowerPoint-cannot-insert-a-video-from-the-selected-file-acd46430-9e0c-4dca-9484-19cf0afdde7c)場合は、問題を解決します。また、[PowerPoint でサポートされているオーディオ ファイル形式およびビデオ ファイル形式](https://support.office.com/article/video-and-audio-file-formats-supported-in-powerpoint-d8b12450-26db-4c7b-a5c1-593d3418fb59)についてもご確認ください。
