# <a name="install-visio"></a>Visio をインストールする

Visio はスタンドアロンアプリケーションとして販売されているため、まだ購入していない場合[は購入するか、お試しください](https://products.office.com/visio)。 

**注**: Visio Online プラン1には、デスクトップ版の visio は含まれていないため、インストールすることはできません。

Visio Online プラン2へのサブスクリプションを持っている場合は、ユーザーに[ライセンスを割り当てる](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1764)必要があります。または、自分の[アカウントポータル](https://portal.office.com/account#installs)に**インストール**から visio をインストールするオプションがありません。 

非ボリュームライセンス版の Visio Standard または Visio Professional は Microsoft アカウントに関連付けられており、管理センターで管理することはできません。 これらのバージョンをインストールするには、ユーザーはこのバージョンの Visio に関連付けられている Microsoft アカウントを使用して[account.microsoft.com](https://account.microsoft.com)にサインインする必要があります。

詳細については、「 [visio をインストール](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1764)する」および使用しているバージョンの visio の手順に関するトピックを参照してください。

詳細については、次のトピックを参照してください。<br>
[Visio の展開ガイド](https://docs.microsoft.com/deployoffice/deployment-guide-for-visio)<br>
[異なるバージョンの Office、Project、および Visio を同じコンピューター上にインストールするためのサポートされているシナリオ](https://docs.microsoft.com/deployoffice/install-different-office-visio-and-project-versions-on-the-same-computer)<br>
[Visio のライセンス](https://products.office.com/visio/microsoft-visio-volume-licensing-visio-for-multiple-users)
