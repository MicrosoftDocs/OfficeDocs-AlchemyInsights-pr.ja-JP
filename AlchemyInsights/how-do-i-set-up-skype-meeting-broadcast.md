# <a name="cant-start-or-join-a-skype-meeting-broadcast"></a>Skype 会議ブロードキャストを開始または参加できませんか?

イベントチームメンバー (会議プロデューサー) として Skype 会議ブロードキャストを開始または参加するときに問題が発生した場合は、構成を再度確認してください。 Skype 会議ブロードキャストには、手動のセットアップ手順が必要です。 この記事の手順に従って、ネットワークと Skype for Business Online ポータルが正しく構成されていること、および影響を受けるユーザーが競合するライセンスによってブロックされていないことを確認します。

[Skype 会議ブロードキャスト用にネットワークをセットアップする](https://docs.microsoft.com/SkypeForBusiness/set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast)
