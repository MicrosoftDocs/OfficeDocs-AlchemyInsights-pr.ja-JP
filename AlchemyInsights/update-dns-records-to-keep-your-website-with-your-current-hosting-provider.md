# <a name="update-dns-records-to-keep-your-website-with-your-current-hosting-provider"></a>DNS レコードを更新して現在のホスティング プロバイダーに Web サイトを維持する

Office 365 パブリックの web サイトを別のホスティング プロバイダーに移動するプロセスの一環として、する可能性が高い必要がありますドメイン レコードを変更します。
  
1. 管理センターでは、**セットアップ**に移動\>**ドメイン**を変更するドメイン名を選択します。 
    
2. 新しいホスティング プロバイダーの web サイトに合った情報を入力します。
    
詳細については[、現在のホスティング プロバイダーの web サイトを保持するレコードの DNS の更新](https://support.office.com/en-us/article/update-dns-records-to-keep-your-website-with-your-current-hosting-provider-2c4cf347-b897-45c1-a71f-210bdc8f1061)を参照してください。 
  
パブリック サイトの販売終了についての詳細については、[中止するのには SharePoint オンライン パブリック web サイト](https://support.office.com/en-us/article/sharepoint-online-public-websites-to-be-discontinued-e86bfd2f-5c7d-446f-a430-7cfcc0130916?ui=en-US&amp;rs=en-US&amp;ad=US)を参照してください。
