# <a name="site-is-not-available"></a>サイトを使用できない

SharePoint または OneDrive サイトが複数のユーザーに対して使用できない場合は、一時的なサービスの問題が発生している可能性があります。 [サービス正常性ダッシュボード](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)をチェックして、テナントに影響があるかどうかを確認します。
