# <a name="site-is-not-available"></a>サイトを使用できない

SharePoint または OneDrive サイトを複数のユーザーが使用できない場合は、一時的なサービスの問題が発生している可能性があります。[サービス正常性ダッシュボード](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)でテナントが影響を受けているかどうかを確認してください。
