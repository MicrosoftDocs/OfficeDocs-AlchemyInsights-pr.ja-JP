# <a name="site-is-not-available"></a>サイトを使用できない

複数のユーザーが SharePoint または OneDrive のサイトを利用できない場合は、サービスに一時的な問題が発生している可能性があります。 [サービスの正常性ダッシュボード](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) を確認して、テナントが影響を受けるかどうかを調べます。
