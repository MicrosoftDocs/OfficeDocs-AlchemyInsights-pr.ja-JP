# <a name="site-is-not-available"></a>サイトが利用可能ではありません。

SharePoint または OneDrive サイトが複数のユーザーに利用できない場合は、一時的なサービスの問題がある可能性があります。[サービス正常性ダッシュ ボード](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)テナントが影響を受けるかどうかを確認するを確認してください。
