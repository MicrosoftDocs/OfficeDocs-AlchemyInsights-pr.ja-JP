# <a name="trying-to-remove-your-domain"></a>ドメインを削除する場合

Office 365 からドメインを削除するには、「[Office 365 からドメインを削除する](https://support.office.com/article/https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx)」の手順に従います。
  
ドメインを削除する前に、そのドメイン名に割り当てられているすべてのユーザーまたはグループを更新する必要があります。他の一般的な解決方法については、「[ドメインに関する FAQ](https://support.office.com/article/https://support.office.com/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx)」を参照してください。
