# <a name="trying-to-remove-your-domain"></a>ドメインを削除しようとしていますか。

Office 365 からドメインを削除するには、次の[Office 365 からドメインを削除する](https://support.office.com/article/https://support.office.com/en-us/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx)手順を実行をします。
  
任意のユーザーまたはドメインを削除する前にそのドメイン名に割り当てられているグループを更新する必要があります。他の一般的な解決策については、[ドメインのよく寄せられる質問](https://support.office.com/article/https://support.office.com/en-us/article/Domains-FAQ-1272bad0-4bd4-4796-8005-67d6fb3afc5a.aspx)を確認してください。
