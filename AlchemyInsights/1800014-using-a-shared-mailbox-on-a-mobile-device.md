# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>モバイル デバイスでの共有メールボックスの使用

モバイル デバイスにインストールされた Outlook アプリから共有メールボックスにアクセスすることはできません。この機能が必要と思われる場合は、[ここで投票](https://go.microsoft.com/fwlink/?linked=862116)することで、Outlook のエンジニアリング チームに知らせてください。
  
モバイル デバイスで共有メールボックスを使用する唯一の方法は、ブラウザーを開いて Office 365 にサインインし、Outlook on the web に移動することです。手順については、「[共有メールボックスを Outlook on the web に追加する](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)」を参照してください。
  
モバイル デバイスで Outlook アプリを使用して共有メールボックスにアクセスする必要がある場合は、共有メールボックスを使用するのではなく、Office 365 のグループを作成することを検討してください。詳細については、「[グループを比較する](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)」をご覧ください。
  
詳細: [iOS 版および Android 版 Outlook のアプリ内ヘルプを利用する](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
