# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>モバイル デバイスで共有されているメールボックスを使用します。

モバイル デバイスにインストールされている Outlook のアプリケーションからは、共有されているメールボックスにアクセスできません。この機能を作成しますか。Outlook のエンジニア リング チームを知ることができます。[投票は、ここ](https://go.microsoft.com/fwlink/?linked=862116)!
  
共有されているメールボックスは、によって、ブラウザーを開いたり、Office 365 にサインイン] をクリックし、web 上で Outlook をモバイル デバイスでのみ使用できます。手順については、 [web 上の Outlook で共有されているメールボックスの追加](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)を参照してください。
