# <a name="setting-up-project"></a>プロジェクトの設定

 Microsoft Project をセットアップするには、[この記事](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281.aspx) の手順に従います。

**Mac ユーザーへのお知らせ:** Project 製品は、現時点で MacOS では使用できません。 
  
セットアップ中にエラーが発生する場合は、[サポート/回復アシスタントを使用して Project をインストール](https://aka.ms/SaRA-ProjectSetupScenario)します。
