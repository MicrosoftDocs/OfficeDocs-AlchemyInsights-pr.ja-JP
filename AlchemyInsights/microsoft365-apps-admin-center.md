# <a name="microsoft-365-apps-admin-center"></a>Microsoft 365 Apps の管理センター

Microsoft 365 Apps の管理センター (config.office.com) は、エンタープライズで Microsoft 365 アプリを展開して管理する管理者向けに、クラウド上で最新の管理を提供しています。
