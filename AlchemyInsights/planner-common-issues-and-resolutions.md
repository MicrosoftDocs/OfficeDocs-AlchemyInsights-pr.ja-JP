# <a name="planner-common-issues-and-resolutions"></a>Planner の一般的な問題と解決策

Microsoft Planner でよく求められるトピックの一部を次に示します。
 
- **作業の開始:** [planner の使用を開始](https://support.office.com/article/microsoft-planner-help-4a9a13c6-3adf-4a60-a6fc-15c0b15e16fc?ui=en-US&rs=en-US&ad=US)し、[組織の planner をアクティブ化する方法、ゲストユーザーを追加する方法、および環境でのアクセスを管理](https://docs.microsoft.com/office365/planner/planner-for-admins)する方法について説明します。
 
- **ゲストユーザー:** Planner への[ゲストユーザーアクセス](https://support.office.com/article/guest-access-in-microsoft-planner-cc5d7f96-dced-4da4-ab62-08c72d9759c6?ui=en-US&rs=en-US&ad=US)を管理します。
 
- **通知:**[受信する通知の種類](https://support.office.com/article/stay-on-top-of-tasks-and-plans-with-email-and-notifications-cce223d6-b0ae-43cf-a080-266e2414a859)、それらをトリガーするアクティビティ、通知の設定を変更する方法を確認します。
 
- **コメント** 
   - Exchange Online メールボックスがないタスクに対する[コメントの表示または追加の問題](https://docs.microsoft.com/office365/planner/planner-for-admins#can-people-in-my-organization-use-planner-if-they-dont-have-an-exchange-online-mailbox)について説明します。
   - [すべてのユーザーが Planner の電子メールを取得できる](https://docs.microsoft.com/office365/planner/planner-for-admins#how-do-i-make-sure-all-my-users-can-get-emails-forplanner)ことを確認する方法について説明します。

- **タスクの管理:**[タスクに関するコメント](https://support.office.com/article/comment-on-tasks-in-microsoft-planner-fd4aedde-7785-4cd0-96ee-122fbc9140e1?ui=en-US&rs=en-US&ad=US)を確認し、[タスクまたはプランを削除](https://support.office.com/article/delete-a-task-or-plan-39e10e78-13f0-446d-94cd-9e562648497a)する方法について説明します。
