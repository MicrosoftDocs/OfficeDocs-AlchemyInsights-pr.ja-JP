# <a name="font-formatting-problems-in-excel"></a>Excel でフォントの書式設定に関する問題

Excel の 2016年のフォントの書式設定が正しく表示されないセルに。これは、複数の種類のフォントの書式がセルに含まれている場合に発生します。たとえばいくつかの文字は、太字で表示される、および他のユーザーは太字で表示される、表示される太字の書式が表示されない可能性があります。または、同じセルに別の色付きの文字があれば、表示されない色が正しく表示されます。エンジニア リング チームは、この問題を認識が修正プログラムを使用しています。その間、別のフォントを使用してください。または、複数の種類のこの問題を解決するまで、同じセル内で書式設定を使用しないようにしようとしてください。
