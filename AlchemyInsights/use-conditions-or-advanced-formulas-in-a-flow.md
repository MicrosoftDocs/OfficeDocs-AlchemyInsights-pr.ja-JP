# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>フロー内の条件や高度な数式を使用してください。

フローをカスタマイズするには、その条件が true の場合にのみタスクを実行できるようにします。情報[フローに条件の追加](https://go.microsoft.com/fwlink/?linkid=872112)を参照してください。
  
高度な数式を作成するには、[ロジックの Azure アプリケーションのワークフローの定義言語スキーマ](https://aka.ms/logicexpressions)を参照してください。
