# <a name="join-a-team-by-code"></a>コードでチームに参加する

Microsoft は、ユーザーが **[チームに参加]** コードを作成できない、または **[コードで参加]** 機能を使用してチームに参加できないという問題を認識しています。 回避策として、チームの所有者は必要に応じて手動でユーザーを追加できます。
