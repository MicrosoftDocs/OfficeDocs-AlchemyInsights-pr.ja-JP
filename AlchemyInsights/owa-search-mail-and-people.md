# <a name="search-mail-and-people-on-outlook-on-the-web"></a>Outlook on the web でメールや連絡先を検索する

1. ページの上部の **検索** ボックスに、検索内容 (連絡先、件名、またはメッセージの一部) を入力し、Enter キーを押します。

2. 検索が終わったら、**検索** ボックスの戻る矢印を選択するか、左側のウィンドウのフォルダーをクリックして、検索を終了します。

詳細については、「[Outlook on the web でメールや連絡先を検索する](https://support.office.com/article/b27e5eb7-3255-4c61-bf16-1c6a16bc2e6b)」を参照してください。
