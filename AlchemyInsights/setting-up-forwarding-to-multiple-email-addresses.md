# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>複数のメール アドレスへの転送を設定する

まず、[[グループの管理](https://portal.office.com/adminportal/home#/groups)] に移動します
  
1. *配布リスト*を作成します 
    
2. 複数のユーザーをそこに追加します
    
3. 次に、そのグループに転送するメールボックスを構成します
