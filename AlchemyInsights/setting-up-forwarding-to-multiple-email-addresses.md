# <a name="setting-up-forwarding-to-multiple-email-addresses"></a>複数のメール アドレスへの転送を設定する

ユーザーのグループへのメール転送を設定する必要がある場合は、配布リストを作成し、それに複数のユーザーを追加して、そのリストへの転送を構成します。 最初に、管理センターで、[**グループ** > [](https://portal.office.com/adminportal/home#/groups)] [グループ] に移動します。
  
1. グループを作成します。

2. 複数のユーザーを追加します。

3. そのグループへのメールボックス転送を構成します。
