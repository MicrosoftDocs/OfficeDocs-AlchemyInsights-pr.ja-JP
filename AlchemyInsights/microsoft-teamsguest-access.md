# <a name="microsoft-teams---guest-access"></a>マイクロソフト チームのゲスト アクセス


有効にするか、マイクロソフトのチームでのゲスト アクセスの役割を構成する必要がある場合があるためのステップ バイ ステップ ガイド:[ゲスト アクセスのチェックリスト](https://docs.microsoft.com/microsoftteams/guest-access-checklist)
  
チーム構成の変更を有効にするには、最大で 24 時間を待機する必要があります。
