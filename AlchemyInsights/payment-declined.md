# <a name="update-or-add-a-new-payment-method"></a>お支払い方法の更新または新しいお支払い方法の追加

1. [**課金情報**] > [**請求と支払い**] > [<a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">支払方法</a>] ページに移動します。
2. [**お支払方法の追加**] を選択するか、更新するカードまたは銀行口座を選択して [**編集**] を選択します。
3. お支払い方法の情報 (カード名義、銀行口座名義、請求先住所、電話番号、有効期限など) を更新し、[**保存**] を選択します。
4. サブスクリプションのお支払い方法をこの新しいお支払い方法に変更してください。 手順は次のセクションで説明します。

新しいカードを追加した後も以前のクレジット カードに対して拒否の通知が表示される場合、その新しいカードをサブスクリプションに関連付ける必要があります。

1. [**課金**] > [<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">製品とサービス</a>] の順に移動します。
2. 新しいカードまたは銀行口座を使用してお支払いを行うサブスクリプションを選択します。 [**課金**] セクションで、お支払い方法の横にある [**編集**] を選択します。
3. 現在のお支払い方法の横にある [**変更**] を選択します。
4. ドロップダウン メニューから別のお支払い方法を選択するか、新しいお支払い方法を追加します。

[お支払い方法を追加、更新、削除する](https://go.microsoft.com/fwlink/?linkid=2118133)
