# <a name="app-configuration-policies-for-microsoft-intune"></a>Microsoft Intune のアプリ構成ポリシー

Microsoft Intune のアプリ構成ポリシーは、iOS または Android アプリケーションの構成設定を提供します。 これらの構成設定により、アプリのカスタマイズが可能になります。 構成ポリシーのサポートは、オペレーティング システムのバージョンとアプリケーションによって制限されます。

詳細については、[「Microsoft Intune のアプリ構成ポリシー」](https://docs.microsoft.com/intune/app-configuration-policies-overview) を参照してください。
