# <a name="create-users-in-hybrid-deployments"></a>ハイブリッド展開でユーザーを作成します。

組織がオンプレミス環境と Office 365 のハイブリッド展開の場合、オンプレミスの Active Directory 内のすべてのユーザー オブジェクトを作成する必要があります。組織の Office 365 に同期されているために、適切な組織単位 (OU) にユーザー オブジェクトを作成します。
