# <a name="help-understanding-value-added-tax-vat"></a>バリューを理解するため税 (VAT) を追加

英国など追加税 (VAT)、値を使用する国の場合は、マイクロソフトの請求は、VAT を計算して金額を請求書に適用されます。
  
税の理由から、請求の設定で、VAT 登録番号を指定することもあります。
  
追加または VAT レートを削除することはできませんが、設定の既定の VAT レートを変更することができます。既定では、請求では、サービスを使用して、VAT 登録番号を検証しますが、欧州委員会によって発行された[VIES の免責事項](https://go.microsoft.com/fwlink/?LinkID=841741)を読むことをお勧めです。サービスを無効にする場合は、すべての VAT 登録番号を確認する必要があります。
