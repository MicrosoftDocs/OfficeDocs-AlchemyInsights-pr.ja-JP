# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>ユーザーのプライマリ アドレスに電子メール エイリアスを作成します。

手順については、[ユーザーの電子メール エイリアスの追加](https://support.office.com/article/0b0bd900-68b1-4bf5-808b-5d240a7739f4)を参照してください。
  
 **注**: 組織は、ハイブリッド モードでは場合、オンプレミスの Exchange 組織のプライマリ電子メール アドレスを変更する必要があります。詳細については、[メールボックスのための追加または削除する電子メール アドレス](https://technet.microsoft.com/library/bb123794.aspx)を参照してください。
