# <a name="missing-clutter-folder"></a>低優先メール フォルダーが見つかりません

メールボックスに [低優先メール] フォルダーが表示さない場合は、[優先受信トレイ] オプションを使用している可能性があります。 詳細については、「[Outlook の優先受信トレイ](https://support.office.com/article/focused-inbox-for-outlook-f445ad7f-02f4-4294-a82e-71d8964e3978)」をご覧ください。
