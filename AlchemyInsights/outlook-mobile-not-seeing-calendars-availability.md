# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>空き時間情報を選択するときに予期した予定表が表示されない

予定表が表示されていない場合は、[予定表] に移動し、画面の左上隅にあるメニューボタンをタップして、予定表が選択されていることを確認します。 送信可能時間を選択するときに、このメニューで有効になっているカレンダーのみが表示されます。
