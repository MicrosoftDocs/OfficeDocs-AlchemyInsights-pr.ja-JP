# <a name="control-creation-of-o365-groups"></a>O365 グループのコントロールの作成

既定では、企業内のすべてのユーザーは、O365 グループを作成できます。必要な場合はを変更し、社内のユーザーのセットに O365 グループを作成する権限を割り当てることができます。O365 サービス (Outlook、チーム、プランナーなど) からのグループの作成に影響を与えるこのことに注意してください。
  
 方法: [Office 365 のグループを作成できるユーザーの管理](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
