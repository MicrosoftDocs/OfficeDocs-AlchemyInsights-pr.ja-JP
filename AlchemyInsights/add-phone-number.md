# <a name="add-phone-number"></a>電話番号を追加する

組織内でユーザーを設定し、電話を掛けたり受けたりする前に、ユーザーの[電話番号を取得する](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)必要があります。 電話システムのセットアップを完了するための詳細なプロセスを提供しました。 「[組織内の電話システムの設定](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview)」をご確認ください。
