# <a name="data-streamer-add-in-for-excel"></a>Excel 用 Data Streamer アドイン

Excel 用 Data Streamer アドインは、すべての Microsoft 365 サブスクライバーに無料で提供されています。 Data Streamer を有効にする方法の詳細については、「[Data Streamer アドインを有効にする](https://support.office.com/article/enable-the-data-streamer-add-in-70052b28-3b00-41e7-8ab6-8a9f142dffeb)」を参照してください。

**注意** Microsoft 365 サブスクライバーの方は、[最新バージョンの Office](https://support.office.com/article/install-office-updates-2ab296f3-7f03-43a2-8e50-46de917611c5) をインストールしていることをご確認ください。

[Data Streamer でできること](https://support.microsoft.com/office/what-is-data-streamer-1d52ffce-261c-4d7b-8017-89e8ee2b806f)  
[Excel 用 Data Streamer アドインの詳細設定](https://support.office.com/article/advanced-settings-for-excel-s-data-streamer-add-in-94cda451-880c-43c7-903c-0212ee188460)  
[Excel 用 Data Streamer アドインのブックの設定](https://support.office.com/article/workbook-settings-for-excel-s-data-streamer-add-in-e9ca60fe-a8ef-4124-8a0a-95df7ba62998)  
[Microsoft Data Streamer をダウンロードする](https://www.microsoft.com/download/details.aspx?id=56976)

Data Streamer アドインについての[フィードバックをお寄せください](https://edusupport.microsoft.com/support?product_id=hacking_STEM&session=9654f308-da1c-4bc2-a6f5-b5faf7a99bbc&auth=1&nf=1&fromAR=1)。
