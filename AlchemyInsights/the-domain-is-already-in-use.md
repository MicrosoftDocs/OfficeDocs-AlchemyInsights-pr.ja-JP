# <a name="the-domain-is-already-in-use"></a>ドメインが既に使用されている

追加しようとしているドメインは、既に別の組織に属しています。対象組織にドメインを追加するには、まず別の組織から削除する必要があります。下に示す記事では、そのプロセスを取り上げます。
  
- [Office 365 からドメインを削除する](https://support.office.com/article/Remove-a-domain-from-Office-365-f09696b2-8c29-4588-a08b-b333da19810c.aspx)
    
- [ドメインを追加して DNS レコードを作成する](https://support.office.com/article/Create-DNS-records-for-Office-365-when-you-manage-your-DNS-records-B0F3FDCA-8A80-4E8E-9EF3-61E8A2A9AB23.aspx)
