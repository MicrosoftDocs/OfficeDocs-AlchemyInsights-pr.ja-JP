# <a name="entity-type-not-showing-in-search-results"></a>検索結果に表示されないエンティティの種類

エンティティタイプを Dynamics 365 統合インターフェースで分類された検索結果または関連性検索結果に表示するには、最初にアプリモジュールに追加する必要があります。

注: 関連性検索で添付ファイルを検索する場合、親レコードが結果に表示されているものであっても、添付ファイルエンティティをアプリモジュールに追加する必要があります。
