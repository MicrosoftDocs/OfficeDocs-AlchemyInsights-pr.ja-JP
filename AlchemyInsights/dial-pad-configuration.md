# <a name="teams-dial-pad-configuration"></a>Teams のダイヤル パッドの構成

Teams クライアントでは、ダイヤル パッドを使用して、公衆交換電話網 (PSTN) 機能にアクセスできます。  

詳細については、「[ダイヤル パッドの構成](https://docs.microsoft.com/microsoftteams/dial-pad-configuration)」をご覧ください。
