# <a name="create-and-use-a-shared-mailbox"></a>共有メールボックスを作成および使用する

共有メールボックスを使用すると、複数のユーザーは、追加のライセンスを割り当てることなく、1 つのエイリアス (sales@contoso.com など) で電子メールを送受信できます。
  
- 始めるには、[こちら](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)をご覧ください
    
「[共有メールボックスを作成する](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx)」の記事では、共有メールボックスに関して必要な情報を、使用方法も含め、すべて確認することができます。
