# <a name="create-and-use-a-shared-mailbox"></a>共有メールボックスを作成および使用する

共有メールボックスを使用すると、複数のユーザーは、追加のライセンスを割り当てることなく、1 つのエイリアス (sales@contoso.com など) で電子メールを送受信できます。
  
- 開始するには、Microsoft 365 管理センターで以下の[手順を実行](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)します。 

「[共有メールボックスを作成する](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)」の記事を読むこともできます。これにより、共有メールボックスについて知っておく必要があるすべての情報がわかります。その使用方法が含まれます。
