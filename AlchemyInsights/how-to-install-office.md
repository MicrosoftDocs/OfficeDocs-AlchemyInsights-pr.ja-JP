# <a name="how-to-install-office"></a>Office をインストールする方法

Office クライアント アプリケーションをインストールするには、Office 365 のビジネス プレミアム、Office 365 のビジネス、または Office 365 の教育などのビジネスのサブスクリプションがあることを確認します。
  
> [!NOTE]
> [オフィス ホームおよびビジネス 2016年](https://products.office.com/home-and-business)は、[自宅用の Office](https://support.office.com/article/28cbc8cf-1332-4f04-9123-9b660abb629e?wt.mc_id=Alchemy_ClientDIA.aspx)製品とビジネスのサブスクリプションの一部ではないです。 
  
- ユーザー アカウントを作成し、自分のドメインを追加します。の[ライセンスが割り当てられている](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc?wt.mc_id=Alchemy_ClientDIA.aspx)ユーザーが持っていることを確認します。
    
- 各ユーザーがあります。
    
1. ポータル ページを参照して: [https://portal.office.com/OLS/MySoftware.aspx](https://portal.office.com/OLS/MySoftware.aspx)。
    
2. サインインしていない場合は、職場、学校のアカウントでサインインします。
    
3. [Office] セクションで、言語を選択します。32 ビット版または 64 ビット バージョンを選択する**詳細設定**を選択します。 
    
4. [ **インストール**] を選択します。
    
詳細な手順とトラブルシューティングのソリューションを参照してください:[ビジネスのための Office 365 をセットアップ](https://support.office.com/Article/6a3a29a0-e616-4713-99d1-15eda62d04fa?wt.mc_id=Alchemy_ClientDIA)します。
