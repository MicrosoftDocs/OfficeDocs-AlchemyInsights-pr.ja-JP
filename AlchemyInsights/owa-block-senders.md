# <a name="block-or-unblock-senders"></a>送信者をブロックまたはブロック解除する

**[受信トレイ] から:**

- メッセージ リストでメッセージを選択してから、**[迷惑メール]** > **[受信拒否]** の順に選択します。

**[設定] から:**

1. ページの上部にある歯車アイコンを選択して、設定を開いて **[Outlook のすべての設定を表示する]** を選択します。
2. **[メール]** > **[迷惑メール]** の順に選択します。
3. **[受信拒否送信者とドメイン]** で、**[追加]** を選択します。
4. ブロックするドメインの電子メール アドレスを入力して、Enter キーを押します。
5. **[保存]** を選択します。

送信者のブロックを解除するには、ブロック解除する電子メール アドレスまたはドメインの横にあるごみ箱アイコンを選択します。

詳細については、「[Outlook on the web で送信者をブロックする、または送信者のブロックを解除する](https://support.office.com/article/9bf812d4-6995-4d19-901a-76d6e26939b0)」を参照してください。
