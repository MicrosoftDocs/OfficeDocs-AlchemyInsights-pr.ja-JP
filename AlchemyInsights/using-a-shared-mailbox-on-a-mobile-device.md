# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>モバイル デバイスでの共有メールボックスの使用

モバイル デバイスで共有メールボックスを使用するには、[Outlook モバイルをダウンロード](https://products.office.com/ja-JP/outlook-mobile-for-android-and-ios) して、「[Outlook モバイルに共有メールボックスを追加](https://support.office.com/ja-JP/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f)」します。
  
ヘルプを表示する: [iOS 版および Android 版 Outlook のアプリ内ヘルプを表示する](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
