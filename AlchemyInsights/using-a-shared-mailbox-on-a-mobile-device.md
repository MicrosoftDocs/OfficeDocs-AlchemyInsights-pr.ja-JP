<span data-ttu-id="63b28-p101">モバイル デバイスにインストールされた Outlook アプリから共有メールボックスにアクセスすることはできません。この機能が必要と思われる場合は、[ここで投票](https://go.microsoft.com/fwlink/?linked=862116)することで、Outlook のエンジニアリング チームに知らせてください。</span><span class="sxs-lookup"><span data-stu-id="63b28-p101">You can't access a shared mailbox from the Outlook app that's installed on the mobile device. <span data-ttu-id="63b28-p102">モバイル デバイスで共有メールボックスを使用する唯一の方法は、ブラウザーを開いて Office 365 にサインインし、Outlook on the web に移動することです。手順については、「[共有メールボックスを Outlook on the web に追加する](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="63b28-p102">You can only use a shared mailbox on a mobile device by opening your browser, signing in to Office 365, and then going to Outlook on the web. <span data-ttu-id="63b28-p103">モバイル デバイスで Outlook アプリを使用して共有メールボックスにアクセスする必要がある場合は、共有メールボックスを使用するのではなく、Office 365 のグループを作成することを検討してください。詳細については、「[グループを比較する](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="63b28-p103">If you must access shared mailboxes using the Outlook app on a mobile device, consider creating an Office 365 Group instead of using a shared mailbox.
