# <a name="learn-how-to-configure-agenda-settings"></a>議題の設定を構成する方法を学習します。

議題の設定は、web 上の Outlook の**予定表**の設定にあります。詳細については、 [web 上の Outlook の予定表の設定](https://support.office.com/article/12cba5a4-4f95-4d00-bfc3-b694aa67ac8f)を参照してください。
  
管理者は、Exchange オンライン PowerShell の**セット CalendarNotification**コマンドレットを使用して、議題の設定を構成します。詳細については、[一連の CalendarNotification](https://technet.microsoft.com/library/dd351284)を参照してください。
