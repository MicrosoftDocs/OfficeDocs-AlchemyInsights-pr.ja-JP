# <a name="would-you-like-to-report-a-spam-false-positive-to-microsoft"></a>迷惑メールについての誤検知を Microsoft に報告しますか?

管理者として、フィッシングまたはスパムメッセージが配信された理由を判断する最良の方法は、サンプルを送信した[エクスプローラー](https://protection.office.com/reportsubmission)で Microsoft に送信することです。 これを easer にするために、ユーザーが[レポートメッセージアドイン](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview)を使用して直接 Microsoft に報告できるようにすることができます。

[Office 365 スパムフィルターをバイパス](https://docs.microsoft.com/exchange/troubleshoot/antispam/cautions-against-bypassing-spam-filters)することを決定した場合は、注意が必要です。
