# <a name="how-to-add-or-reduce-licenses"></a>追加またはライセンスを削除する方法

削除するか、ライセンスを追加購入するには。
  
1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. 左側の [ライセンスの追加または削除サブスクリプションを選択します
    
3. [ **ライセンスの追加/削除**] を選択します。
    
4. 必要なユーザーの合計数を入力します。
    
    たとえば、25 のライセンスがある現在、5 を削除する場合は、このフィールドに「20」と入力します。
    
5. [ **Submit**] を選びます。
    
> [!NOTE]
> 任意の新しい費用 (ライセンスの増加) またはクレジット (ライセンスの減少) に表示されます、次回の請求。
