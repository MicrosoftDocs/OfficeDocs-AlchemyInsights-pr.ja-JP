# <a name="create-a-new-folder"></a>新しいフォルダーを作成します。

1. ナビゲーション ウィンドウの [**フォルダー** ] の下には、**新しいフォルダー**を選択します。 
    
2. フォルダーの名前を入力し、Enter キーを押します。
    
詳細については、 [Outlook.com でフォルダーを作成する](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)を参照してください。
