# <a name="use-microsoft-edge-extensions"></a>Microsoft Edge 拡張機能を使用する

[拡張機能](https://go.microsoft.com/fwlink/?linkid=2135619) は Microsoft Edge にインストールできるアドオンです。広告のブロックや、パスワードの管理、テキストの翻訳など、さまざまな方法で閲覧環境をカスタマイズできます。
