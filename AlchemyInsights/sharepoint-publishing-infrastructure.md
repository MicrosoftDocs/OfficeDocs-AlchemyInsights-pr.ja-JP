# <a name="sharepoint-publishing-infrastructure"></a>SharePoint 発行インフラストラクチャ


SharePoint には、従来の機能セットが含まれており、イントラネットのリッチ web ページを作成して発行することができます。 SharePoint 発行インフラストラクチャをサイトコレクションレベルでアクティブ化し、オンにします。

詳細については、以下のリンク先を参照してください。

[発行機能の有効化](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[発行機能](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

注:

- 従来の発行サイトテンプレートには、既定で発行機能が含まれています。

- モダンサイトでは、従来の発行機能を使用できません。 モダンサイトは、既定では新しいものを提供しています。
