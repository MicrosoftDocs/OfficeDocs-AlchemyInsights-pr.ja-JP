# <a name="banned-sending-ip"></a>禁止送信 IP

Microsoft 365 にメールを送信するときに、このエラーが送信者に表示されることがあります。 送信者は https://sender.office.com にアクセスして、指示に従う必要があります。  ポータルからの確認メールを送信者が受信していない場合は、送信者にスパム フォルダーをチェックしてもらいます。または、別のメール アドレスで試してもらいます。
