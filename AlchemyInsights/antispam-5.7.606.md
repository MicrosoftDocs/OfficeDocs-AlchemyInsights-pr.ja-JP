# <a name="banned-sending-ip"></a>禁止送信 IP

Office 365 の受信者に送信するときに、このエラーが送信者に表示されることがあります。 送信者は [[Office 365 スパム対策用 IP リストから除外ポータル](https://sender.office.com/)] にアクセスし、除外する必要のあるメール アドレスと IP アドレスを提供する必要があります。 ポータルからの確認メールを送信者が受信していない場合は、送信者にスパム フォルダーをチェックしてもらいます。または、別のメール アドレスで試してもらいます。 

- 詳細な手順については、「[リストから除外のポータルを使って、受信拒否リストから自分自身を削除する](https://docs.microsoft.com/microsoft-365/security/office-365-security/use-the-delist-portal-to-remove-yourself-from-the-office-365-blocked-senders-lis?view=o365-worldwide)」をご覧ください。
