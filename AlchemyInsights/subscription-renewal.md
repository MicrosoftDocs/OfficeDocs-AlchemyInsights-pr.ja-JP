# <a name="subscription-recurring-billing"></a>サブスクリプションの定期的な請求

毎月または毎年のサブスクリプションが (**定期的な請求**がオンになっている) 場合に請求するか、(**定期的な請求**がオフになっている) 場合、期限切れを知りたいかどうか。 
  
1. 管理センターで、[[サブスクリプション](https://go.microsoft.com/fwlink/p/?linkid=842054)] ページに移動または**請求先**を選択\>**のサブスクリプション**。
    
2. 詳細を知りたいサブスクリプションを選択します。
    
3. サービス カードの右側にある、サブスクリプションの状態、**定期的な請求書**の設定、および次の請求額や有効期限日に関する情報です。 
    
4. この日以降のサービスを使用して続行するには、**上**に**定期的な設定と課金情報**の表示/非表示を切り替えます。サブスクリプションの次の請求日は、ステータスが表示されます。
    
5. この日にサービスを停止するには、**オフ**に**定期的な請求書**の表示/非表示を切り替えます。ステータス、サブスクリプションの有効期限が切れる日付が表示されます。
