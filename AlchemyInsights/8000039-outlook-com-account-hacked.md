# <a name="outlookcom-account-hacked"></a>Outlook.com アカウントのハッキング

自分のアカウントがハッキングされた場合は、[マイ Outlook.com アカウントがハッキングされた](https://go.microsoft.com/fwlink/p/?linkid=874366)ことで、アカウントを復元するのにはの指示に従います。 
  
[Office 365 のサブスクリプション サーバーの Outlook.com の高度なセキュリティ](https://go.microsoft.com/fwlink/p/?linkid=874368)の詳細を表示します。
