# <a name="learn-about-accessibility-in-microsoft-edge"></a>Microsoft Edge のアクセシビリティの詳細

Microsoft Edge では、最新の [UI オートメーション API (UIA API)](https://go.microsoft.com/fwlink/?linkid=2153423) を導入しました。 UIA への変更は、ブラウザーのアクセシビリティへの主要な投資であり、Windows 10 の支援技術に依存するユーザーのためのより包括的な Web エクスペリエンスの基盤を築きます。 

詳細については、「[Microsoft Edge のアクセシビリティ](https://go.microsoft.com/fwlink/?linkid=2153512)」を参照してください。
