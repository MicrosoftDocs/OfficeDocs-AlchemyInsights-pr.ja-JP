# <a name="configure-multi-factor-authentication-in-office-365"></a>Office 365 の多要素認証を構成します。

MFA を構成するには、 [Office 365 のユーザーに対して多要素認証の設定](https://support.office.com/article/8f0454b2-f51a-4d9c-bcde-2c48e41621c6.aspx)を参照してください。
  
有効にして、MFA を構成するのには、Office 365 のグローバル管理者である必要があることに注意してください。
