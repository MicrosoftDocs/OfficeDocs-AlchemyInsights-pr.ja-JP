# <a name="delete-a-folder"></a>フォルダーを削除する

1. 右側のウィンドウで、削除するフォルダーまたはサブフォルダーを右クリックし、**[削除]** を選択します。 
    
2. 続行するには **[OK]** を選択します。 
    
詳細については、「[Outlook.com でフォルダーを削除する](https://go.microsoft.com/fwlink/p/?linkid=873134)」をご覧ください。
