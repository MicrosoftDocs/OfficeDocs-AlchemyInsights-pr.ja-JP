# <a name="delete-a-folder"></a>フォルダーを削除します。

1. 左側のウィンドウで、フォルダーまたはサブフォルダーを削除し、[**削除**] を選択するを右クリックします。 
    
2. 確認するのには **[ok]** を選択します。 
    
詳細については、 [Outlook.com でフォルダーを削除する](https://go.microsoft.com/fwlink/p/?linkid=873134)を参照してください。
