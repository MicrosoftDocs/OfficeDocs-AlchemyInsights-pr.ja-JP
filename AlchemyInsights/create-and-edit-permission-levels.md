# <a name="working-with-sharepoint-permission-levels"></a>SharePoint アクセス許可レベルを使用する

SharePoint には、**投稿**や**表示のみ**など、使用できる既定のアクセス許可レベルが付属しています。 ただし、これらのいずれかがニーズに合わない場合は、新しいアクセス許可レベルを作成できます。 既定値を変更しない場合は、アクセス許可レベルをより簡単に追跡できる点に注意してください。

[アクセス許可レベルを作成および編集する方法](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
