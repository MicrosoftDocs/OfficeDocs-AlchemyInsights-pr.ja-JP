# <a name="cant-join-teams-meeting"></a>Teams 会議に参加できない

トラブルシューティング方法  

- 管理センターの **チャネルの今すぐ会議の使用を許可する** ポリシーオプションが無効になっている可能性があります。 確認するには、Teams管理者に問い合わせてください。
- Google Chrome でTeams　web アプリを開き、会議に参加を試みてください。
- [Teamsでインスタント会議を開始する](https://support.microsoft.com/office/start-an-instant-meeting-in-teams-ff95e53f-8231-4739-87fa-00b9723f4ef5)にある「ユーザーの招待」の手順に従って、会議に参加している人から招待をしてもらうこともできます。
- 画像が破損していないことを確認します。
