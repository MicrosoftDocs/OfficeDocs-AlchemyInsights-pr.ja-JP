# <a name="sync-warning-last-synced-more-than-3-days-ago"></a>同期の警告: 前回の同期が 3 日以上前です

その他のトラブルシューティングについては、「[Office 365 のディレクトリ同期の問題を解決する](https://docs.microsoft.com/ja-JP/office365/enterprise/fix-problems-with-directory-synchronization)」を参照してください。

「[Office 365 でディレクトリの同期状態を表示する](https://docs.microsoft.com/ja-JP/office365/enterprise/view-directory-synchronization-status)」の手順に従って、Office の管理ポータルで同期状態を確認することができます。
