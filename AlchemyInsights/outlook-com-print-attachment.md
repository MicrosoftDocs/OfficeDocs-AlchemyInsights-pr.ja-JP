# <a name="print-an-attachment"></a>添付ファイルを印刷する

1. 印刷する添付ファイルが含まれるメッセージを開きます。
2. 添付ファイル上にカーソルを移動して、**[ダウンロード]** アイコンを選択します。
3. ファイルがダウンロードされた後、適切なプログラムを使ってファイルを開き、印刷オプションを選択します。たとえば、Word で .doc ファイルを開いて印刷します。

詳細については、「[Outlook.com でメール メッセージ、添付ファイル、予定表を印刷する](https://go.microsoft.com/fwlink/?linkid=2021110&amp;clcid=0x409)」を参照してください。
