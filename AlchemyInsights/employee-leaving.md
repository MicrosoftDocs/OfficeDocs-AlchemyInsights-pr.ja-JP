# <a name="have-an-employee-leaving-and-need-to-block-access-to-data-and-email"></a>まま従業員があるし、データや電子メールへのアクセスをブロックする必要がありますか。

これを実行するには、次の操作を実行します。
  
1. [アクティブなユーザー](https://support.office.com/article/https://portal.office.com/adminportal/home.aspx#/users)に移動します。
    
2. ブロックしユーザーを選択します。 
    
3. サインイン状態の横にある [**編集**] をクリックします。 
    
4. **サインインからユーザーをブロック**を選択して\>**を保存**します。 
    
 **注**: 一部のアプリケーションをすぐできない場合がありますのサインアウトします。この資料内の他のアクションをフォロー アップ: [Office 365 からの元の従業員を削除します。](https://support.office.com/article/https://support.office.com/en-us/article/Remove-a-former-employee-from-Office-365-44d96212-4d90-4027-9aa9-a95eddb367d1.aspx)
