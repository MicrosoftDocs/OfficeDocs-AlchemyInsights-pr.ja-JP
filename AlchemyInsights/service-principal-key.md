# <a name="service-principal-key"></a>重要なサービス プリンシパル

サービス プリンシパルの主要な資格情報管理のガイダンスについては、次の記事を参照してください。

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) - PowerShell を使用してサービス プリンシパルの重要な資格情報を取得します。

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) - PowerShell を使用してサービス プリンシパルの新しい重要な資格情報を作成します。

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) - PowerShell を使用してサービス プリンシパルから重要な資格情報を削除します。
