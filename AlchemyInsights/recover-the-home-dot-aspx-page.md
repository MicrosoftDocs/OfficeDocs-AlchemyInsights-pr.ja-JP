# <a name="recover-the-homeaspx-page"></a>Home.aspx ページを復元する

サイトのホームページが削除された場合は、ごみ箱の URL にアクセスするとごみ箱から復元できます。

1. <**tenantname**>をテナントの名前、<**sitename**>をサイト名に置き換えて、https://\<tenantname>.sharepoint.com/sites/\<sitename>/_layouts/15/RecycleBin.aspx? に移動します。

2. 復元するページを選択します。

それでもホームページが見つからない場合は、新しいサイトページを作成して、[サイトのホームページとして設定 ](https://support.microsoft.com/en-gb/office/use-a-different-page-for-your-sharepoint-site-home-page-35a5022c-f84a-455d-985e-c691ab5dfa17?ui=en-us&rs=en-gb&ad=gb)することができます。 サイトにアクセスしようとしたときに、"404（ページが見つかりません） " エラーが表示されなくなります。
