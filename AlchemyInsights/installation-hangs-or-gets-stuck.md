# <a name="installation-hangs-or-gets-stuck"></a>インストールがハングアップするか、スタックを取得

コンピューターのネットワーク接続、ウイルス対策ソフトウェア、または Windows Update の設定によってこの現象を参照してください可能性があります。ここでは、いくつかの点が問題を解決しようとすることができます。
  
- コンピューターを再起動し、Office がインストールされているかを確認します。
    
- ワイヤード (有線) 接続を使用します。
    
- 低速のインターネット接続の場合は、インストールを Office の[Office オフライン インストーラーを使用します。](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA.aspx)
    
- ウイルス対策ソフトウェアを一時的に無効します。
    
- コントロール パネルからオンラインの修復を実行します。
    
- 削除し、Office を再インストール
    
[Office をインストールするのには時間がかかり](https://support.office.com/article/0f09f357-3fef-42a6-b8aa-cef4c6c44bdf?wt.mc_id=Alchemy_ClientDIA.aspx)
