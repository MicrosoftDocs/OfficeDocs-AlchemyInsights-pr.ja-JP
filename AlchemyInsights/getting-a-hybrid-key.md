# <a name="getting-an-exchange-hybrid-key"></a>Exchange ハイブリッド キーの取得

オンプレミスの組織を Microsoft 365 の Exchange Online に接続するには、オンプレミスの組織に少なくとも1つの Exchange サーバーを設定する必要があります。このサーバーは、オンプレミスの Exchange 組織を Exchange Online に接続するハイブリッド展開を作成するために使用されます。このサーバーのハイブリッドキーを取得するには、「 [Exchange ハイブリッドプロダクトキー配布ウィザード](https://aka.ms/hybridkey)」の手順を実行します。
