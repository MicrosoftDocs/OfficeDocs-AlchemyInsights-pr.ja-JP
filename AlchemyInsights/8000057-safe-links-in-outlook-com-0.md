# <a name="advanced-outlookcom-security-for-office-365-subscribers"></a>Office 365 サブスクライバー用の Outlook.com の高度なセキュリティ

Outlook.com をご利用の場合、スパムやマルウェアのフィルタリング機能を活用できます。Office 365 Home か Office 365 Personal に加入している場合、Outlook.com ではさらに、受信したメッセージの添付ファイルやリンクに対して追加のスクリーニングが実行されます。
  
Web ページのリンクを含むメッセージを受信すると、Outlook.com では、そのリンクがフィッシング詐欺に関連しているか、ウイルスやマルウェアがお使いのコンピューターにダウンロードされる可能性があるかがチェックされます。
  
詳細については、「[Office 365 サブスクライバー用の Outlook.com の高度なセキュリティ](https://go.microsoft.com/fwlink/p/?linkid=2006140)」を参照してください。メッセージから安全なリンクを削除する方法も記載されています。
