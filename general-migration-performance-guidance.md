# <a name="general-migration-performance-guidance"></a>全般的な移行パフォーマンス ガイダンス

移行のパフォーマンスは、ネットワーク インフラストラクチャ、ファイル サイズ、移行時間、調整の影響を受ける可能性があります。これらの要因を理解することは、移行効率を計画し、最大化する上で役立ちます。
